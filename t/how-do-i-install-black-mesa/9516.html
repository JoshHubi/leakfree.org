In my downloaded package (torrent) there is an EXE file I simply had to run (exit steam!). Then simply fire up steam and play it or change some startup properties for BM, I have to use: -dxlevel 90 -w 1280 -h 1024 -novid -heapsize 512000 on my Debian system.

Always change -w (width) and -h (height) to your screen resolution if you play on wine to have lesser (not entirely bug-free, but very good playable) bugs.

Cheerios August 11, 2020, 9:57am #14

  1. Be sure to install Steam https://store.steampowered.com/Steam

  2. Be sure to install Source SDK 2007. From inside the Steam interface click “Library”, find a dropdown menu that is default labeled as “All Games” and in that dropdown, choose “Tools”. Scroll towards the bottom of the list to find and install the “Source SDK 2007”

  3. Close down and exit Steam

  4. Download the game
    https://www.gamefront.com/black-mesa-is-live-download-it-here/
    Scroll down to see three separate download links. Download all three.
    a) blackmesa.7z
    b) BlackMesa.md5
    c) blackmesa-setup.exe

  5. Install the game by clicking on blackmesa-setup.exe and be sure it is installed to this directory:\Steam\steamapps\sourcemods\BMS

  6. Launch Steam and find the game “Black Mesa” listed in your game’s “Library”
