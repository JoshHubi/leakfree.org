Where in the config can i fix this or is there a spare config someone has.

Add “-w {WIDTH} -h {HEIGHT}” to your launch options of “Black Mesa” inside the Steam Library (All Games).

How-To add or change Black Mesa’s launch options:

  1. Open Steam
  2. Browse to the “Library”
  3. Select “All Games”
  4. Right-Click “Black Mesa”
  5. Click “Properties”
  6. Click “Set Launch Options…”
  7. Type “-w {WIDTH} -h {HEIGHT}” (without “”)
  8. Click “OK”
  9. Close the “Black Mesa - Properties” window
  10. Start the game by clicking “Play!”
{WIDTH} = 1920
{HEIGHT} = 1080

Note: “1920” and “1080” are just examples, you can use whatever (valid) values you need.

Valid values would be:

1920x1080

-w 1920 -h 1080

1680x1050

-w 1680 -h 1050

1600x900

-w 1600 -h 900

1440x900

-w 1440 -h 900

1280x1024

-w 1280 -h 1024

Note 2: Be sure to remove the added commands and values “-w {WIDTH} -h {HEIGHT}” from the launch options, after you’ve changed the resolution in-game, using the options panel (Video).
