If you are trying to install the mod version (I believe that’s the ,legacy mod, you’re talkin’ about), simply download the torrent file on moddb, run the executable (It’ll automatically find the Steam/steamapps/sourcemods folder) and install.

You also need to have SDK Base 2007 installed from Steam’s Tool tab, and after this, you’re ready to go.

Moddb page: https://www.moddb.com/mods/black-mesa/
