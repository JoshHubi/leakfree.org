robot_deodorant November 25, 2018, 1:41am #46

I have been using Windows 8 as my sole OS since the first public beta (DP) came out months ago just because I could.

At first I really regretted it, and came close to rolling back to Windows 7. When the newer beta (CP) was released it fixed a lot of problems I had with it, and really I have no problem with Win8 anymore. It is a little faster than 7, it’s easier in some areas, such as the new task manager (which actually helps more than I thought it would), the new Windows Defender is awesome, it’s nice to have a built-in AV… so really the most annoying thing for me is the start screen.

The Start screen is annoying. Whenever I logon, it’s there. Everytime I want to run a search, it’s there. Everytime I go to click on the first icon in my taskbar, I accidentally click on the Start popup instead… and BOOM. The start screen likes to fling itself on top of whatever you’re doing and won’t go away until you find the “Desktop” tile and click it.

It happens about 3+ times a day on accident, of course you actually do need to click the Start button if you do need to access the Start screen. And that’s when the start button doesn’t want to appear… >

As annoying as it is, you get used to it, it becomes muscle memory to click the Desktop tile, and actually some of the metro apps have been fun and useful. :s hrug:
