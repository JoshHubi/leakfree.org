BTW, you guys saying Firefox 4 is consuming more memory than Chrome, I disagree:


https://download.cnet.com/8301-2007_4-20047314-12.html

I ran a small test here. I disable all extensions from both browsers and opened Chrome’s tab “about:memory” and measured the amount of private memory that was used.

2 tabs
Firefox: 114 MB
Chrome: 99 MB

3 tabs
Firefox: 145 MB
Chrome: 118 MB

4 tabs
Firefox: 144 MB
Chrome: 127 MB

5 tabs
Firefox: 149 MB
Chrome: 147 MB

6 tabs
Firefox: 183 MB
Chrome: 195 MB

7 tabs
Firefox: 192 MB
Chrome: 205 MB

This test makes me think that FF uses more memory for the browser and less memory for each tab when compared to Chrome. However, extensions also use a lot of memory in Firefox.

BTW, performance-wise, Firefox 4 is losing for Chrome only in benchmarks related to Javascript, which is not where speed is that necessary.

Regardless of anything, I have to say that Chrome is an incredible browser which got a lot of popularity and forced all other browsers (including IE) to change for the best, both in performace and appearance. Firefox 4, IE 9 and Opera Whatever only exist because of Chrome. But this time my vote goes for Firefox 4
