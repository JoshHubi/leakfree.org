Solve the following differential equations and receive a prize;

Find the equation Z(x,y) given the differential equation
dZ(x,y) = [y^2+Cos(x)]dx + [2xy + Sin(y)]dy = 0

hint: Use Partial derivatives

From here

Solution

px will be used as partial derivative
dx will be used as derivative

[y^2+Cos(x)]dx + [2xy + Sin(y)]dy is in the form pZ/px * dx + pZ/py * dy = 0

Step 1] Lets see if this is an exact differential, so let us define two variables
pZ/px = m = y^2 + Cos(x)
pZ/py = n = 2xy + Sin(y)

Now if they are exact differentials then taking pM/py should equal pN/px

pM/pY = p/py(y^2) + p/py(Cos(x)) = 2y + 0
pN/pX = p/px(2xy) + p/px(Sin(y)) = 2y + 0

We have a winner, they are the same, so we can use the exact differential method to solve.

2] Lets integrate to find our function z with partial derivative with respect to x

Let z = integral of m * dx = integral of y^2 + cos(x) dx = xy^2 + sin(x) + H(y) = C1

3] H(y) is the left over junk from partial derivatives, so lets get rid of this

Lets use a partial derivative of xy^2 + sin(x) + H(y) with respect to y and set it equal to pZ/py(n) to find the derivative of H(y).

pZ/py = n = 2xy + Sin(y) = 2yx + H(y) = 0 so H(y) = sin( y )

integrate both sides with respect to dy to find H(y) = -cos(y) + C2

4] Lets insert H(y) back into the function and let C3 = C1 + C2

z = xy^2 + sin(x) - cos(y) = C3
