russilker November 24, 2018, 11:14pm #2

Even if you put it in and it doesn’t work, nothing should be damaged. It just won’t POST.

Before you pop it in, though, upgrade the BIOS of the computer with the old CPU (if you still have it), and if you really want to make sure it’ll work, nothing wrong with calling Dell support. They should be able to tell you exactly.

EDIT: But yeah, it should work. From what I read online both incorporate the same socket type, LGA 775.

Winged_One November 24, 2018, 11:14pm #3

How would I upgrade the BIOS with the old CPU?

russilker November 24, 2018, 11:14pm #4

Download it from Dell.com (there should be a section for drivers and software, make sure you get the one for your specific PC model). AFAIK you can update it through Windows, but if not you’d need to put it on removeable media and boot into it while starting up the computer.

That’s as specific as i can get without reading up on your PC, you should be able to figure it out. But try to pop in the new CPU if you already have it, maybe a BIOS update isn’t necessary
