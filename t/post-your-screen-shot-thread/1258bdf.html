Try checking the “3D resolution” setting in the options. ARMA 2 has two resolution settings - one for the interface (which determines the resolution the game runs at) and another for the 3D graphics (which determines the resolution the 3D assets and effects are rendered at). You might have the Interface res set to 1920x1200 but the 3D res set to something much lower, thus resulting in blur when it gets upscaled.

Dreadmoth November 25, 2018, 12:39am #4608

It’s not quite useless - you get the benefit of a sharper in-game interface without as much of a performance hit as you would get if everything was rendered at the higher resolution.
There’s not much point using it if your computer can run the game well already at your screens native resolution, but it can be useful on lower-end computers (as long as the limiting factor in performance is the graphics card, not the processor… otherwise it provides very little benefit at all).

PS: It also works in the other direction: you can render the game at 200% of your normal resolution to get some crazy super-sampling going
