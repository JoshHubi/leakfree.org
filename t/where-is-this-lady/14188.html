Crypt:

IIRC most if not all trailer NPCs have been replaced.

Also, not that it really master’s bit I think that lady is white, lighting in still is just funky.

100% sure of this, as the trailer was made before the Face Creation System was implemented. Though it would have been nice to have had some head submodels based off the trailer ones to use with the FCS, for extra variation.

As for racial variation among the female staff, There are some additional skins for the female scientist model. but they never appear in game because there aren’t any face presets using them - likely a result of the fact that you don’t encounter a very large amount of female scientists in the game.

It’d be nice if they were used in some of the random definitions so they could appear in custom maps, but even doing that, the non-white skins wouldn’t appear during the main game because IIRC every female scientist in the game uses their own specific preset face.
