Given:
The derivative of ArcTan(Q) is 1 / ( 1 + Q^2 )
Proof

  1. Substitute Q^2 for x in the original equation, so we get
    1 / ( 1 + Q^2 ) = 1 - (Q^2) + (Q^2)^2 -(Q^2)^3 + (Q^2)^4 - (Q^2)^5 +…
    1 / ( 1 + Q^2 ) = 1 - (Q^2) + (Q^4) - (Q^6) + (Q^8) - (Q^10) +…

  2. Integrate both sides
    ArcTan(Q) = Q - (Q^3)/3 + (Q^5)/5 - (Q^7)/7 + (Q^9)/9 - (Q^11)/11 +…

  3. Let Q = 1, so ArcTan(1) = PI/4, &
    ArcTan(1) = 1 - (1^3)/3 + (1^5)/5 - (1^7)/7 + (1^9)/9 - (1^11)/11 +…
    PI / 4 = 1 - 1/3 + 1/5 - 1/7 + 1/9 - 1/11 +…

PI = 4 * ( 1 - 1/3 + 1/5 - 1/7 + 1/9 - 1/11 +… )
PI = 4 * Sum of [ (-1)^n * 1 / ( 2n + 1 ) ] where N starts from 0 to infinite
