In some of the older versions of catalyst there was an option that enabled catalyst to display a logo in the corner of any game that uses crossfireX to show the the user that the game is using that feature. This option was implemented back when very few games utilized multiple gpu’s, crossfireX or SLI, and has been removed in newer catalyst’s. You can get the latest version here.

I just found where to turn off the setting. Right click the ATI logo on the notification bar and u should see something that says ATI Radeon 3800 series, put ur mouse over that and something should appear that says crossfirex settings. Put ur mouse over that and u should see somethin called Show CrossfireX status icon. If it has a checkmark next to it, then click it and that should disable the ATI logo.
