Information From Steam Forums Tutorial

Portal Blockable walls

To make a wall that blocks a portal from being paced on it, create a brush entity and make it a func_noportal_volume which will come up invisible in game; so give it the invisible texture so you can see what’s behind it.

Also

How to make walls unportalable:

  1. Create a new brush
  2. open the texture browser and search for no draw
  3. apply the no draw texture to the brush
  4. select the brush and press ctrl + t
  5. change its class too func_noportal_volume
  6. place it in front of a wall that you dont want a portal to be made on.

The textures determine if a portal can be used or not and the dimensions of the texture area.

In the .vmt:

Textures not usable by portal gun require
“%noportal” 1

LarsBR November 24, 2018, 8:32pm #3

An example vmt-file from Portal:

“LightmappedGeneric”
{
// Original shader: BaseTimesLightmap
“$basetexture” “Metal/metalwall060a”
“$surfaceprop” “metal”
“%keywords” “airexchange,combine”
“%noportal” 1
}

The important part is the “%noportal” 1
