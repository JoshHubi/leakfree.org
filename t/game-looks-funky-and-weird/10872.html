Put it simple, there are two major ways to download the game:

  1. Official download mirrors.
  2. Torrent.

Choose any of these, there are no other options (expect for finding a friend who has the game downloaded and copying installation files from him).

As for the game looking like in screenshots: it could be anything starting from software/hardware related problem and ending with corrupted game data files.
Redownload/reinstall. If the problem persist - try updating the drivers. Game is known to work normally (except for some performance issues - but that’s something everybody suffers from to some extent) on the hardware like you have. BTW, how much VRAM your GPU has? You hadn’t posted it and I remember that there were some low-cost video cards having next-to-none VRAM installed on them and using main system RAM as a storage space for textures. In case you have such - it could be your problem cause.
