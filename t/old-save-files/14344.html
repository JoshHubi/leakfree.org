TextFAMGUY1 November 25, 2018, 4:36am #3

Old saves will definitely never work. Even saves from previous patches on the retail version do not work properly.

If you want to start a new game from whatever chapter you were on in the mod release, you can use the console command:

sv_unlockedchapters 15

On the main menu, and this will unlock all chapters. You can then start a new game from whatever chapter you were on before.
