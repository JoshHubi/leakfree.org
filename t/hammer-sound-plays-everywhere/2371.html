RabidMonkey November 24, 2018, 7:18pm #2

Are you using an ambient_generic or just telling the func_rotating to make a sound? If you’re doing the latter, try using a looped ambient_generic to play your sound, as you have more control over the sound radius. If you’re using an ambient generic make sure not to set a SourceEntityName as that tends to cause problems with sounds playing everywhere (also make sure the ‘play everywhere’ flag isnt ticked)
