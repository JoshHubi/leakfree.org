DinoThrasher November 24, 2018, 7:00pm #446

You need to use the following commands:

fog_override 1
fog_enable 0
fog_enableskybox 0

I don’t know why, but all those commands require sv_cheats to be on. I guess because having the fog turned off can affect gameplay? I hope they eventually remove the sv_cheats lock, because this would be fun to play with.
Edit: This doesn’t work in daylight levels. It still creates a black sky off in the distance which doesn’t look right in the daylight, and while keeping the skybox turned on with fog off creates nice, sharp shadows, it also makes the draw distance look weird since the game’s draw distance is designed to be hidden behind the fog effect.

Also, for anyone who doesn’t know, if you add these two commands to your launch options:

+mm_dedicated_force_servers 1
+mm_dedicated_search_maxping 150

It prevents the game from creating a local server during server searches as well as making sure the dedicated server you find doesn’t give you more than 150 ping. I think the default for that command is 300.
