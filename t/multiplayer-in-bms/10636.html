Roby65 November 25, 2018, 2:26am #3

It’s simple. (you need the console)

Type in the console:

net_start
maxplayers 4
sv_lan 0
map [map name]

The client, should open the console and type:

connect [ip]

Naturally replace the map name with the real map name (bm_…) and the ip with your ip.
If you get stuck somewhere (very probably) type:

sv_cheats 1
noclip

If you find how to change player model, tell me
SetModel seems not working…

Roby

PS: Also if the second player gets stuck without HEV suit, he needs to write

give item_suit
