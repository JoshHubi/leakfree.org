In case you guys don’t know, when you build a game for 360 using Visual Studio XNA, it automatically creates a Project that makes the game compatible for Windows (or at least, there is a setting for that). The resources should be the same (since XNA supports mp3s, jpegs for textures, etc) so I would imagine only the logic should change (physics, AI, etc) due to CPU differences.
