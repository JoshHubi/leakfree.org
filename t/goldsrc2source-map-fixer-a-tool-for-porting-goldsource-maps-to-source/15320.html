GOLDSRC2SOURCE - Map Fixer: A tool for porting GoldSource maps to Source

Hello
I created a tool for porting your GoldSource maps to Source. So far it features many fixes that will save hours of mindless work.

How to use[/size]

Click the Find VMF button and find the vmf file you want to repair. You can choose what fixes do you want the tool to perform. Unless you know what you’re doing, I recommend leaving them all checked. When you click Repair, the program will appear to freeze. This can take anywhere from 50 seconds to 2 minutes or more depending on the size of your map.

This tool is designed to modify .vmf files. Using it with other map formats like .map, .rmf or .jmf might result in a crash or a broken map.

Not all features are yet implemented. So far, this tool should fix most of the problems, but additional work will still be required. When compiling the map in Hammer be sure to uncheck HDR. Otherwise textures will appear to be glowing.
I recommend clicking the Help button to see a full manual and an explanation of what does what.
