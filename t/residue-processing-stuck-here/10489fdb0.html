There are some difficult jumps in our game, but it overall shouldn’t be that tough. No, suit power doesn’t affect jump distance. I wonder if this is as a result of a bug (which we thought we fixed a long time ago, but may not have succeeded in doing so).

Could you check something for me, if you know how? Could you please open the developer console, while the game is running (use the tilde key, normally to the left of the 1 key on your keyboard), and type

sv_gravity

Then press enter. It will tell you what the game’s value for sv_gravity is set to. The value should be 600. If it is 800, then our bug has not been fixed, and we’ll need to look at it again! We had a weird bug before we launched the game last year, where gravity would keep setting itself to 800, when it was meant to be 600. This makes jumping considerably harder, as you get way less breathing room on all the jumps. It wouldn’t make the game unbeatable, just much harder in an unfun way.
