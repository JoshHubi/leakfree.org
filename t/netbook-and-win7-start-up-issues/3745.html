russilker November 24, 2018, 8:48pm #5

If you get into safemode, in the Run box (the search box in the start menu), type in MSCONFIG

Go into the Services and Startup tabs and copy down all of the entries, and post em up here. We can tell you which you can disable so your bootup experience is a minimal one. Then when that works (and it should), you can enable them one-by-one or something to figure out the issue.
