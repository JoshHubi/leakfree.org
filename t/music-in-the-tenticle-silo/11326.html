Jenova19 November 25, 2018, 2:41am #6

There is a console command to adjust the music volume specifically:

snd_musicvolume "<VALUE>"

Note: Replace with any (valid) value between 0 and 1.0.

With 0 being 0% and 1.0 being 100% volume.

Just add snd_musicvolume “” to the autoexec.cfg and see if it works, in-game.

Don’t forget to replace , though.

EDIT (03:07 AM): This is what the console says about snd_musicvolume:

Console:

[COLOR=‘DarkRed’]“snd_musicvolume” = “1.000000” ( def. “1.0” ) min. 0.000000 max. 1.000000

Note: Use either the ‘find snd_musicvolume’ or the ‘help snd_musicvolume’ commands (both without ‘’), to recreate said quote.
