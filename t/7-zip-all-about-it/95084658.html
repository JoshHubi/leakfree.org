It’s ok, I figured it out. For anyone else who has had this problem, (because I’ve seen in asked in a few other forums as well): When I downloaded it the first time, it crashed. So I downloaded it again, and my computer automatically named the file “blackmesa.7z(1)”. That’s why the setup couldn’t recognize it. Just right click the file and rename it to “blackmesa.7z”. BOOM!
