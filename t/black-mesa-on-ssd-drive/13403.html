From what I know, source mods must be installed in respective Steam subfolder so that Steam can notice and run them. But Black Mesa, consisting of a lot of small files, would probably load and run much faster on a SSD drive. So is it possible to install it to another location and still launch and play it?

Dragunov2 November 25, 2018, 4:13am #2

Of course, just create a steam folder on the other drive

Xeoxer November 25, 2018, 4:13am #3

But My SSD drive is not big enough to house all my Steam games, it’s only 100 GB and a large portion of it is already occupied by Win7. So I wanted to know if it’s possible to install a single Steam mod somewhere else.

Crypt November 25, 2018, 4:13am #4

Steam > Settings > Downloads > Steam Library Folders > Add Library Folder

You can have multiple libraries on multiple drives, they don’t all need to be in one.
