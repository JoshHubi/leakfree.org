I had a lot of failed installations with code 2 errors. This was due to the BlackMesa.7z file being corrupt. The following tells you how to repair using a torrent program but without the hassle of a full new download.

Put the corrupt BlackMesa.7z file into the directory where your torrent program downloads files (I use BitTorrent) and then download the torrent from the download section of Black Mesa website. As you already have the BlackMesa.7z file your torrent program will assume you have already partially downloaded the file and start to check if the file is complete and ok. In my case it found it to be 99.5% ok and downloaded an additional 400mB . The extra download repairs the corrupt file - not sure why it was corrupt as like the OP I had downloaded loads of times and could not get one to work.

Once the download completed I copied it (The repaired BlackMesa.7z file) into the recommended location \Program Files\Steam\steamapps\SourceMods\BMS along with the BlackMesa-Setup.exe file.

This time installation worked with no errors and game is ready to play - you will need to quit and restart Steam before it shows up in your library.
