Marb November 24, 2018, 8:42pm #8

I doubt you’ll find many servers using v3.0 anymore. There’s downloads for it somewhere, but I’d suggest sticking with the latest.

So if you do plan to stick with 4.07, try this - I’ve had some weird issues and this seems to fix most odd problems (not including known bugs and other issues like the whole fmod thing).
Assuming you’re running the Steam version of HL:

  1. uninstall Sven Co-op
  2. uninstall Half-life (save any files you wish to keep first)
  3. delete Half-life folder (may be optional, but that’s what I did just to get rid of everything)
  4. restart Steam and reinstall Half-life
  5. once HL is at 100%, run one game of HL and exit out
  6. download and install SC 4.0b
  7. download install the 4.07 update
  8. set up your options and you should be good to go

Some of these steps may be completely unnecessary, but I did them all just to be sure and haven’t had any major problems since.
