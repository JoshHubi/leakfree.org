The skin is these two with this pack. The pack is outdated, there’s a version with up to 5 skins out now, but that’s the one I was using at the time of that screenshot.

I should clarify that the skin doesn’t have anything to do with whether NVG’s will show up or not. If you want NVG’s on stock BM grunts and don’t want to change anything, then copy the below text into your character_manifest under the npc_human_grunt. If you have anymore questions you can PM me so we don’t derail this.

	npc_human_grunt
	{
		flex_data
		{
			brow_h 1.0
			neck_size 0.75
		}
		bodygroup_data
		{
			gasmask_nv 1
			gloves 0,1
			holster 1
			packs_chest 0,1
			packs_hips 0,1
			packs_thigh 0,1
		}
		model "models/humans/marine.mdl"
		skin 0,2
	}
