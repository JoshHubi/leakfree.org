Artlav November 25, 2018, 2:26am #4

Assuming default setup and 32bit windows, the game should unpack to
C:\Program Files\Steam\steamapps\sourcemods\BMS
The name should be exactly BMS, or it won’t work.

Going back to C:\Program Files\Steam, this is where you should find the ClientRegistry.blob file.

If your windows is 64bit, then look in “Program Files (x86)” instead.
