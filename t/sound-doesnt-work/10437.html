On my Mac Mini 2011 2.7 Intel Core i7, ATI Radeon 6xxx chip, 8Gb memory it works flawlessly. I have it at 2540xsomething (native for Mac 27" Cinema Display)and it goes fairly smooth. I was using Win8 in the VM. I like how the game runs in it’s own Space and can swipe to and from it. I may have to drop the resolution a little if things get busier and get framerate hits but no complaints so far.

If I run at that resolution on PlayOnMac using Wine 1.5.19, it runs like a dog (have to turn of GLSL for Radeon graphics also, else is unusable with Wine, but Wine plays good at about 2140x something with GLSL off). Fonts in Wine look a bit off and best to run X-Widows full screen, and after exiting hit Opt-Alt-A to see the desktop (else get Black screen)

I think Wine doesn’t have as efficient DirectX emulation as Parallels.
