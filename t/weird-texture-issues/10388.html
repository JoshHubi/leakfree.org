Looks similar to the already reported bug that had turned out to be a GPU drivers problem for some older ATI mobile videocards. If you’re on ATI/AMD - make sure you have latest driver version installed. Don’t rely on drivers ability to “check for update online”, go to the amd site instead and manually download/install the latest available version of the driver from there.

If you’re on nVIDIA - could be hardware overheating/fault issue. What you see on screenshots is not a texture problem, it’s a vertex corruption problem really making the triangles game world is constructed from look weird.
