LordDz November 25, 2018, 4:52am #2

How to add custom textures:

Use VTFEdit, import the texture of your choice, save it as .vtf, in a folder you want your textures to be in the blackmesa materials folder.
Simple, put them in a folder called polardog at your desktop.
in VTFEdit go to Tools -> Generate .vmt

Copy the polardog folder into your bms materials folder.
If you open the .vmt in Notepad++ you can see the path it has set for the texture.

Reopen hammer editor to get the new textures.
Repopen black mesa to get it to use the new textures.

Your Hammer is probably loading below DX9/Shader Model 2.0, which we stripped out support for in the last update, for optimisation.

In your graphics card control panel, ensure that Hammer is set to use your graphics card, and not integrated graphics. This will resolve the issue for sure.
