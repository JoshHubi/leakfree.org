“PC slowdown”
To avoid the problem you have to take control of running programs. And have to try to run one at time. After loading your computer programs are set on start up automatically. Now the Question is how can I see the programs which is running? To see this:
-Open the start menu and go to run options you can see there is a Dialog Box.
-Writer here “msconfig” and press Enter. Then you can see the series of tabs.
-Click to startup. Here you can see a list of programs that set to run when your boots. You can select / Deselect the program that application should run or not by the check box.
-Make you selection and click ok. Your computer may be ask you to restart. If does, after starting a massage will display that telling you that you have change the start up configurations.
