russilker November 24, 2018, 9:28pm #17

Manual registry cleaning is just going through the registry and seeing entries to programs you uninstalled a while ago, for instance, and removing them.

All the program does is speed up the process; it goes through all of your registry entries, checks out what the entries point to, and if it points to nothing the entry is useless, and it removes it. That’s one of the scenarios.

Think of it this way:
In your local library, there is a cabinet filled with cards. Each card has information about one library book. You can go through all the cards manually and trash those that refer to books no longer in the library, or you can use a program to quickly sort through all of them, even if not as efficiently.

Protip:
Manual registry cleaning is most useful when you have some conflicting registry entries/a very disorganized registry, where a program removing a “useless” entry can make another program stop functioning properly. Or, when you only want to remove one specific entry.
