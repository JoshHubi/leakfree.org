I suppose you can always try the standard “shit happens” procedures.

Restart the game.
Check for file integrity.
Delete the game folder (not the GCF).
Delete both the game folder and GCF.

Remember, in doing the last one you’ll lose your saves. They don’t appear to be stored in the game folder itself, so it’s possible that the GCF is modified when you save. Are you using Half-Life or Half-Life: Source?

Also, concerning the renderers, are you talking about the DirectX/OpenGL options? I’m not sure about any alternatives. Running a Google search doesn’t seem to turn up any results, however.

Good luck, sir.
