Mostly I would like the jumping mechanics to be made more consistent with Half Life 2. I know this has been covered in entire topics prior to this, but it seems ridiculous for pretty much every jump in the game to require crouch jumping.

I like crouch jumping. It is a fun way to make jumping to higher levels either easier, in that the bottom of the player’s collider doesn’t catch at all on the platform where it would with a normal jump, or indeed possible at all. However I don’t like having to crouch jump to get onto knee height platforms when I can fairly easily jump up to around chest height without crouching in Half Life 2.

Yes, I could probably fix this myself by changing some jump values in the console, but I’m exceptionally lazy and don’t really see why jumping has been made so awkward in the first place. Plus I don’t know if this would have an impact on the level design of some areas, perhaps allowing me to skip sections with superior jump height, so it might be beyond simple variable changes to make it work well…
