I read about how it was making people’s systems boot up slower. I
originally bought a ssd to run windows, simply because I was getting

What the hell?

I run on windows 7 64 bit premium. Can’t wait for windows 10 because
then I can enable fast boot. Right now, it’s seconds from starting the
computer and landing right into windows. Need that so I can quickly get
into doing work when I’m in a hurry.

At least I got one thing positive to say from using Razer Synapse 2.0.
It made me frustrated enough to go out and buy a ssd that now makes my
computer perform better than anything else I had used before. Now I know
why Linus Tech Tips says, always have a computer with windows installed
on a ssd. Literally makes your computer fun to enjoy. Now enjoying that
speed more so without Razer Synapse 2.0. Love the mouse btw, feels
great in my hand, black light, and it doesn’t have any loud clicking
noises. Really well built mouse. Too bad about the software though. So
glad I didn’t buy one of their stupid mmo razer gaming mouses.
