Jim_DiGriz November 24, 2018, 10:37pm #3

Before resetting your MB, make sure your operating system of choice isn’t set to “hibernate” or “standby” There also might be a setting in the BIOS for “soft off by PBTN” under power management, make sure it’s set the way you want.

BaTz281 November 24, 2018, 10:37pm #4

Following suit with the OS settings (i.e. your start menu default might be Hibernate and not Shutdown) be sure your BIOS settings are set to S3 (or S1&S3). If it’s S1 only, your tower will stay running.

Otherwise - Operating System? 32 or 64 bit? Hardware specs? Connected devices?
