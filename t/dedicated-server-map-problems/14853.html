Not having the dedicated server installed, I can’t confirm this, but according to https://steamdb.info/app/346680/depots/, the current build of the Black Mesa Dedicated Server was released on May 14, 2016 – 22:19:04 UTC. That was the same day the Black Mesa v0.3.0 Content Update was released. However, the v0.3.1 patch to Black Mesa was released on June 5, 2016 – 05:53:03 UTC, and it doesn’t look like they updated the Dedicated Server along with the regular game. So that might explain why the local and server maps differ.
