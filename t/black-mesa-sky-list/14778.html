[align=center]Black Mesa Sky List[/size][/align]

After seeing a few requests for an official sky list on my travels, I thought I’d compile this list, I would put it on the wiki but I can’t get authorized due to a bug and felt I could put it here in the meantime, this list is not complete just yet as I’ve only gone through the vmf’s supplied in[/size]
the retail release, so these values maybe updated by the Crowbar Collective as they get more features in, if a sky is updated then I’ll update the list by amending, not by removal, this allows people to have choice when it comes to a starting default (updated values will be marked in orange[/size])
[/size]

I’ve tried to construct this list to the SourceSDK wiki design principles, I have not been able to create full images of the skyboxes, so each thumb is currently 1 face of the skybox, if you know how to recreate the entire sky for thumbnail use it would be greatly appreciated.
If you like what I’m doing here and want to have more information on a webpage instead of trawling through vmfs for information, I’d be happy to pull up more information on Black Mesas default values for it’s scenes like fog, tonemap etc then give me a shout, I’d love to flesh this out.[/size]
