I have Vista Ultimate and MS .NET Framework 3.5 SP1 installed just fine.

Couple of things to check for, Vista Ultimate SP1 DVD comes with .NET Framework 3.0, so make sure it is installed first by checking the “Windows Process Activation Service” in “Turn Windows Feature On or Off” in “Programs and Features” in Control Panel.

As I recall, once I had .NET Framework 3.0 up and running. installing .NET Framewotk 3.5 (and then 3.5 SP1) was a snap. Never installed .NET 2.0 SP1 or any older .NET frameworks.
