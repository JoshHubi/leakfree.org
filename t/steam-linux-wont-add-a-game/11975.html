d9001 November 25, 2018, 3:15am #2

EXE files are Windows executables (they don’t work natively on Linux.) That EXE file is a Windows installer for Black Mesa. Black Mesa isn’t currently (probably not for a while, if ever) compatable with Linux. If you want to run Black Mesa under Linux, you have to install a Windows compatability layer such as Wine. Once you install Wine, download Steam for Windows and install it, install Black Mesa with that EXE, restart Steam for Windows and it should be under your library. However, it would be better to just use actual Windows to run it for various compatibilty reasons.
