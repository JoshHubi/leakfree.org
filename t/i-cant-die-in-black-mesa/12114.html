Mr_Someguy November 25, 2018, 3:15am #13

For the record, sv_cheats is reset every time you start the game, but not reset if you load another save where it wasn’t on. god and buddha mode (+ things like noclip and notarget) don’t reset, but don’t carry over to other savegames. So if you save a game where god mode is on, god mode will stay on in that save game until you turn it off (if you reload that save you’ll need to turn it off again, but if you turn it off and save, it’ll stay off in the new save).

Also, when sv_cheats is set to 1 and you type in god or buddha for the first time, you’ll get a message in the console under it saying “god mode ON…” or “buddha mode ON…l”. If you type it again, you’ll get “god mode OFF…” or “buddha mode OFF…”, which is what you’ll be looking for here.
