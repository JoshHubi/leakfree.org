If you need it - you can download the free Finale Viewer app here:

https://az18268.vo.msecnd.net/applications/Reader/2011/FinReaderWin.exe

This allows you to print the sheet & also have your PC ‘play thru’ the written notes to help you learn it.

Install the program, then FILE - IMPORT the musicXML file, posted by Slytax24 above.
(I had a few errors about ‘distance’ on the xml import, but just click thru them, it doesn’t affect the sheet music)

If you play it it thru with the viewer, I found that a speed of about 80bpm sounded about right (the player defaults to 120!)

Slytax points out a minor mistake in the Bass in bar 6 - it should read the same as the very last bar.
(and if you are really picky you can open up the XML file in something like Notepad++ and edit line 937 to change the Aflat to a F, and remove the <alter -1> tags above & below to change it to a proper F whole note before you import & print it)

Big Thanks to Slytax - I searched the web for hours for this sheet - it doesn’t exist!
