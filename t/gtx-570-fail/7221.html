GConstantine November 25, 2018, 12:20am #16

Sometimes HDTV using HDMI is configured using 1080p/24p, which is 24fps. If you have V-sync enabled, and the nvidia drivers detect and use 1080p/24p, crysis will lock to 24FPS. Try it with any monitor (other than TV) and see if there is a difference.

Or you could also disable V-Sync and see. If you get max 24FPS, at any setting, then this is probably the problem.
