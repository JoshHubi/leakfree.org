Interloper update

Finished x_c4a1e, x_c4a1f and updated x_c4a1cd.
VMF files are included.
x_c4a1_cd_e_f.zip

To make map conversion easier, I wrote a small AutoIt script that works on decompiled HL Source vmf‚Äôs. It will rename a number of entities to their BMS counterparts and do some advanced stuff on some entities, like transforming keyvalues into I/O for multi_manager (which becomes a logic_relay).
Otherwise, only the most important entities for Xen are currently supported, but support for more entity classes can easily be added.
You will need AutoIt installed to run the script.
Maybe someone finds it useful to get started on some of the other maps.
