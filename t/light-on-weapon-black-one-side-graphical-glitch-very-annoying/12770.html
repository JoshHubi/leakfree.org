Go into the console by pressing the tilde (~) key. If a window that says “console” doesn’t show up, then go to options -> keyboard, then click the ‘Advanced’ button. Click the check that says enable developer console.

Type in the ConVar “sv_cheats 1”. After doing that, type in “r_flashlightscissor 0”.

This should disable the cutting of flashlight shadows. The purpose of the command is to save up CPU/GPU memory. You’ll have to re-type the commands in every time you start the game.
