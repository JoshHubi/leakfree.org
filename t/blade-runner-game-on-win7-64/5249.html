Sky25 November 25, 2018, 2:00am #20

Ok. And now a little fix for the “Garbled screen problem” (and maybe other issues) without the need to use the “ALT+TAB” keys: Search in Google for the “bitpatch ddwrapper” and download the file “ddwrapper.zip”.
Extract ddwrapper.zip in the Game’s directory and edit the file “aqrit.cfg” with notepad or wordpad.
Leave all the values to the right with an “1” except “Wine_Diablo” and “NovideoMemory”, there you should put a “0”.
This works for Windows 7 (Ultimate) but should work also in other versions of W7 and Vista.
