Technology and Gaming
sIMson August 11, 2020, 7:59am #1

On November 22, 2011 John Carmack finally released Doom 3 source code. It has been tweaked a bit to avoid any legal issues. Just like it has been with Quake III Arena source code release, project iodoom3 has been announced.

Looks like we can expect some kickass games based on Doom 3 source soon
