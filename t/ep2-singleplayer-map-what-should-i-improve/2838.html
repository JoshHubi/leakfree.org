Mangled November 24, 2018, 7:47pm #2

The terrain is nice but the grass sprites look out of place on the hard stony areas on the ground. Judging from what little brushwork I can see on the tunnel in the first screenshot you need to work on some detailing for it. I’m guessing it’s supposed to be a mine entrance.

You could use some more features like a cabin/hut to break up the open and repetative environment, perhaps with some ammo/health to find there. You should also have a think about the purpose of your map. If it’s a mine where do the tracks lead? Where would mine carts be unloaded? Are there any buildings nearby where the miners would rest?

Make sure you reference your architecture to real-world examples to help them structurally make sense. The entrance with the 52 above it sticks out like a sore thumb, what era is it supposed to be from? Mines from the early 20th century are mainly built from timber with some steel for reinforcement.
