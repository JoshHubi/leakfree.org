I’ve been able to track down what a few of the specific problems with the categories are. If you create categories and add games to them, do 2 things:

1st = when adding games to a category, add the categories in alphabetical order (first select “half-life” category, then “favorites” then “non-steam”
If you were to add a game to the category “non-steam” and then add it to the “favorites” category this would result in steam forgetting the “non-steam” category because it alphabetically follows the category you selected last.

2nd = restart steam TWICE if you have made ANY changes to what categories games and mods are in. Steam Beta usually forgets the changes on the first restart, then remembers them again on the second restart.
