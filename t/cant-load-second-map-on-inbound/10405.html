BoosterGold November 25, 2018, 2:24am #4

Solution found: Reinstall Steam to something other than the default Program Files folder on Windows 7, preferably “Games”. BMS is unable to pull the player.save files due to Windows 7’s various security issues.

You should be good to go after that.
