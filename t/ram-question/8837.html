CPU August 11, 2020, 8:49am #2

CL stands for CAS Latency. It’s a measurement of time needed to address the address column of the memory (Column Address Strobe). Installing CAS6 spec memory will mean that all of your memory will run at CAS6 rather than CAS5.

You can get what you’re looking for here:
