Try this:

https://arma2.co/files/index.php?dir=Black%20Mesa/

IMPORTANT: Make sure the files have the following MD5 hash values:

502B417A7857C3FFB82A8B2C93EE1ECF *blackmesa-setup.exe
F1D86575816F4C6994A5657EEBF76409 *BlackMesa.7z

There are some sites that have the setup .exe and the .7z archive together in a .zip archive. Don’t use those files because the MD5 hash for the .7z archive is incorrect.
