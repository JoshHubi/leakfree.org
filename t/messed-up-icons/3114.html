sersoft November 24, 2018, 8:07pm #6

Looks like you’re running windows XP, missing icons are not hard to fix, because microsoft hasn’t deleted all the nessesary menus like they did in vista and 7.

Just right click on the desktop, click properties, in the Desktop tab click Customize Desktop, then select the missing icons and click “Restore Default”.

Then press Start, right click on Outlook and press “Remove from this list” (I doubt you’re using outlook, if you are, you shouldn’t be)

The only program I know that can easily manage all your system icons is IconPackager, you could try installing that and apply the windows default package and see if it restores your original icons.
