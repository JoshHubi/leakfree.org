Broken Headcrab physics on zombies

Black Mesa Archive In-game Issues
te47 November 25, 2018, 4:36am #1

Whenever a zombie’s Headcrab is both killed and detached from the zombie, it always falls straight to the floor as if dropped. This is especially awkward when using a shotgun to blow the Headcrab off of a zombie. I believe Half-Life 2 used additional code to artificially create a big physics impulse any time this happens, and this code is apparently missing or broken in Black Mesa. This is one of very few remaining issues that’s been bothering me in the latest version of the game, and it would be a greatly appreciated fix. I think not having the Headcrab come off at all would be preferable to it falling straight down through the zombie every time.
