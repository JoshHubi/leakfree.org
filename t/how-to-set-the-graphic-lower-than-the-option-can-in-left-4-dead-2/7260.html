sersoft November 25, 2018, 12:22am #13

But I haven’t played any L4D games on old computers, I know that portal 2 doesn’t have the dxlevel command anymore, and that some graphics cvar’s need SV_CHEATS 1 to be entered first, you could try to open the game, before you connect to a server change the resolution to 640x480 and type

sv_cheats 1
mat_dxlevel 80
mat_forcehardwaresync 0
mat_antialias 0
mat_specular 0
r_drawdecals 0
r_lod 5
mat_picmip 5
r_shadows 0
r_3dsky 0
dsp_off 1
snd_mixahead 0.5
snd_async_fullyasync 1
cl_forcepreload 1
sv_forcepreload 1

and if it still lags after this,

mat_showlowresimage 1
mat_filtertextures 0
mat_filterlightmaps 0
