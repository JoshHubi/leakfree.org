Sign up for an account at hostgator, install wordpress and live happily ever after.

Forget the basic html coding stuff, static html sites died out a long time ago - especially if you going to be updating on a regular basis. Static sites are good if you have less the 5 or 10 pages. A lot of people say, “oh, I wont be updating ever often”, then they start making post every week. Before you know it you have dozens of pages, and noway to maintan them.

Start off with a basic content management system - CMS - you might be a lot happier later on.
