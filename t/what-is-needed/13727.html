TextFAMGUY1 November 25, 2018, 4:24am #2

Steam is required to play. You do not need any purchased games to play Black Mesa. From Steam, you have to download the Source SDK Base 2007 base. It is under the “tools” section on Steam, and is free for anyone to download and install. Once you have installed the Source SDK Base 2007, you can then install Black Mesa. You must then run Black Mesa from Steam to play it. If you’re uncomfortable with Steam, once you’ve installed everything, you can run it in offline mode, if so you desire.
