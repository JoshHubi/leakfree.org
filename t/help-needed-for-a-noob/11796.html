ac3d August 11, 2020, 9:58am #2

If your download has these files:
BlackMesa.md5
BlackMesa-Setup.exe
BlackMesa.7z

and you have made sure Source SDK Base 2007 is installed and up to date, close Steam, you have two ways to install:

Easy: Run BlackMesa-Setup.exe and follow the instructions

Manual:

  1. Install WinRAR https://www.rarlab.com/download.htm (just download the top x86 32-bit one if you don’t know which one to use)
  2. On your computer, go to (yoursteamfolder)\steamapps\sourcemods
  3. In the “sourcemods” folder, create a new folder named “BMS”
  4. Click the Start Menu, type “Winrar” and press enter
  5. In WinRAR, click File, Open Archive, find BlackMesa.7z and double-click it
  6. Click the Extract To button near the top
  7. In the Extract To window, find the BMS folder you created in step 1 and double-click it, then click OK
  8. When WinRAR has finished extracting, close it
  9. Run Steam
  10. Hopefully Black Mesa will be in your Games List now

ac3d November 25, 2018, 3:00am #4

Do you have the three files in the same location?
I was able to produce the same error message as yours by moving BlackMesa.7z to a different location of BlackMesa-Setup.exe, the three install files listed above can be placed anywhere on your computer, but they must be in the same location together in order for the setup to work properly, you could create a folder called “Black Mesa Install” and move these three files into that folder to ensure this, and then try running the .exe

If for some reason you still get the error message, I suggest you go for the manual install, the game will be listed in Steam and play the same either way.

Make sure Steam, and any torrent applications are closed while attempting the easy or manual install

The last and worst case scenario, during the extraction in the manual install, if WinRAR displays any error message, then the files are incomplete or corrupted, and Black Mesa must be re-downloaded
