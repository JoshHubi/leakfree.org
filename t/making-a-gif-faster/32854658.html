Even if you have typed in one somewhere on a program the outputted gif dose not contain that information. Gifs are drawn differentially. Encoded into each frame is a frame delay. This delay tells the renderer (in this case the browser) how long to wait until displaying the next frame. So yes it is possible to manually set a frame delay of .001 between the first and second frame while having a frame delay of .2 between the next BUT for an animation to run at the desired 10 fps you would set a frame delay of .1 for every frame.
