Corsair Vengeance M90 Mouse ~$69.00

This is the new mouse i’m using.

Best feature that I think you will like is that the mouse wheel is very heavy. In fact the wheel is a cylinder of aluminum. Other cool features include 6 profiles, and 3 on the fly speed setting that you can configure. Plus a neat sniper button that on command sets the DPI to a set level and returns once released. (Handy in Photoshop for precision editing as well as head shots)

Buttons/macros/profiles can be reconfigured in the manager p1 p2 p3

I’ve been happy with the results of it so far. I purchased it a few months ago.
