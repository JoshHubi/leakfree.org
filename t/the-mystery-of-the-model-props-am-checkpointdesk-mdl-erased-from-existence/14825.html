The mystery of the model "props_am\checkpointdesk.mdl" - erased from existence?

General Development Source 1
KevinJRattman August 11, 2020, 11:34am #1

So, in my map there is a desk using model “props_am\checkpointdesk.mdl”.

If anyone’s wondering, it’s the desk with blue battry on it, bottom right. This picture is from the old version of the map, taken circa late 2015.

I was shocked to find that Hammer was unable to find this model this morning.

At first, I assumed they (developers) changed the model name. Sadly, I looked around the model viewer for quite a long time, primarily props_am folder. Alas, no trace of model could be found in the model viewer.

It’s not like I changed the model name recently either; my old VMF file from April 3rd 2016 still point toward this model. This VMF is from before the Surface Tension update. (pic below)

The bms_models_dir.vpk also do not contain said model. (pic below)

The recent Surface Tension patch notes do not mention the removal of this model. I had assumed the model was removed from the game without mention. I mean, such things happen. However, things got more interesting when I decided to roll back to old version of Black Mesa (and get the model file out).

This models VPK file is one from 2015, pre-December Update (CU2, if I remember correctly). Surprisingly, there still is no trace of this model. What’s going on?

I am quite baffled. How is it possible the model is not in 2015 version of the game, if I indeed had played the game and could see the model well back then?

I’m not computer expert - I’m just a humble mathematician. If anyone can shed a light on this problem, please help me solve this mystery. Computers are weird. Maybe I’m missing something. Maybe answer is very childish-ly easy. But I really do not get what might be the problem.

P.S. Yeah, I could just use another model for the desk. But the mystery’s been holding me off from the work. Beside, as a level designer you know how much it is painful to change structure of the room after it’s been settled down.
