Raminator:

You shouldn’t have quotation marks around each shader parameter, you shouldn’t be using vertexlitgeneric unless it’s a model, you shouldn’t have spaces in your file names, and your selfillumination mask should be in your basetexture’s alpha channel.

Your .vmt should probably look like this:LightmappedGeneric { $basetexture screen_hall/screen_hall $normalmap screen_hall/screen_hall_normal $selfillum 1 }

Does it make a difference if it’s a prop?
