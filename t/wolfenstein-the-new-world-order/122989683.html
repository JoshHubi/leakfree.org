Dreadmoth November 25, 2018, 4:11am #358
Kairouseki:

Some of the console commands are locked too which doesn’t make it easy.

You can bypass the lock on some of them by using them together with the cvarrandom command, e.g.:

cvarrandom r_skippostprocess 1 1

Replace “r_skippostprocess” with whatever cvar you’re trying to alter (some have to be set when the game starts or require a renderer restart to apply correctly).

I think the numbers after the command define the range of random numbers the cvar might be set to - “1 1” is only 1, “0 0” is only 0, “0 100” might be a number in the range of 0 - 100
