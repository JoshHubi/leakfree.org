Siggs August 11, 2020, 3:41am #42

I’m uploading some files now, because it’s easier than rolling all around FPSbanana to find what I want

Step 1. Install HL2-Update. Should turn up in the sourcemods folder.

Step 2. Install the Fakefactory outdoors addon into the HLupdate files. NOT NORMAL HL2

Step 3. Install the 2.12 Fakefactory the same way.

Step 4. Downloadmy pack from here.

Step 5. Install the HD citadel and HD arms in my pack. Doesn’t matter which first.

Step 6. Install VertThrashers Weapons pack. THEN install the e3_AR2 afterwards, so it overwrites the one in thrashers pack. (This is personal preference, I like this AR2, the combine assault rifle, more than thrashers. He changed the colour scheme on his)

voila!

p.s. This is my personal preference btw, you may not think the same.
