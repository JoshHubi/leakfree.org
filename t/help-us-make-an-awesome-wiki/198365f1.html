ok, I got the footer template working. This code is placed at the bottom of pages that are all contained in the same category. This allows users to jump from one related item to another.

For example you can jump from media release 4 to media release 11 without needing to go back to the parent category media releases. This is important because most wiki users don’t get to sub categories from there parents, but instead from a link. This also makes navigating the sub categories much simpler.

To use it

{{Foot|TITLE|1• 2• 3 • 4 • 5 • 6 • 7 • 8}}
