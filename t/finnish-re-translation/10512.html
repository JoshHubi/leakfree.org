Finnish captions for Black Mesa: Source

Version 1.0 by kyyrma (with thanks to Matoking)

This release includes full captions for:

  • All NPC dialogue
  • Environment sounds (when using Full captioning)
  • Chapter titles
  • Menu options
  • Achievements

In total the game should be 100% in Finnish.

Known issues:

  • Opening tram ride audio, some npc lines and radio chatter don’t trigger captions (Problem with BM itself)
  • Some small typos and mistakes are still to be expected

Installation:

  1. Navigate to .\Steam\steamapps\sourcemods\BMS\resource\
  2. Backup bms_finnish, closecaption_finnish.txt and closecaption_finnish.dat
  3. Extract the provided .zip into the folder, overwrite when asked

Feel free to provide feedback and corrections!

Download: bms_finnish_10.zip
