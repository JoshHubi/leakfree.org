I have seen a few suggestion to disable gibs for this crash, you might try this.
violence_hgibs 0
violence_agibs 0
https://steamcommunity.com/app/362890/discussions/0/282992562606326574/#c1843493219424430782

In case your saved game or autosaved game is corrupt try the following.
Restart the chapter by picking an unlocked chapter from the New Game menu.
Or you could just restart the individual map.
Type status in console, make note of the map name (example bm_c2a4c ). Then type “map bm_c2a4c” into console. This will start the map as a new game.
