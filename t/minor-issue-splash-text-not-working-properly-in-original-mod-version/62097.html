This issue happens when the mod is not installed in a folder called BMS in the sourcemods folder. There is no need to edit game files. Just make sure you install the mod to <Steam library>\steamapps\sourcemods\BMS. The sourcemods folder should be located in the Steam library location where you installed the Source SDK Base 2007.
