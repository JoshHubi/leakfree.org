I can’t open the console!. I have checked the “enable developer console” box but the problem is that the tilde command has no effect with my swedish keyboard. I have tried to rebind the “toggleconsole” command in the config.cfg (I usually bind it to F11) but it changes back when I start the game.
In HL2 and BM mod version, it was possible to set this in the options menu.

dogelition_man November 25, 2018, 4:27am #2

Workaround:
Launch the game with the -console launch option and in the console, do “bind [key] toggleconsole”

Thank you! It worked!
