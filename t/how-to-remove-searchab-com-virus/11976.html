alexjohnson073 November 25, 2018, 3:41am #4

Searchab.com is a browser hijacker. This virus program gets installed onto the system when user tries to download and install free software or add on. Ask your friend to pay attention when installing software online, because often a software installer (when not downloaded from a trusted source) includes optional installs (which are malicious), such as searchab.com.
Your friend needs to follow the following instruction:
• Firstly let her uninstall the searchab program
• Remove the virus residual damage from the browser.
• In next step she needs to remove searchab.com registry keys with Adwcleaner.
• Run a computer scan with any antimalware software such as MalwareBytes.
• Again scan the whole system for viruses with existing antivirus and if not having any then go for Kaspersky, Immunet or Avast.
Hope this will help.
