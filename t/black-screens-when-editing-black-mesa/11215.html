JeffMOD November 25, 2018, 2:40am #11

There’s a thread around here somewhere on it, but if you don’t feel like searching you could do what I did before I saw that thread;

Make a mod under the Source 2007 configuration. Name it whatever, you’re just going to delete it. Use any available option besides Source Code Only. Refresh the list by switching engines to any other build and back.
Then open the edit game configurations menu, and manually create one for BM. Delete the configuration for your temporary mod, as well as the folder structures the SDK made for it.

Open Hammer, set up the FGDs and your default preferences, and test it.
