What browser does one need to have to use the HTML5 player on vimeo.com? Just curious 'cuz the latest Iron (Chrome) doesn’t seem to do it. Haven’t tried the only other browser I have atm (IE6 :trollface

It seems Vimeo uses H.264 (~ mp4) video format. Iron should not be able to play those videos because it does not have the decoder for patent issues, although Chrome has. The decoder for WebM (Google’s video format) is in Chrome dev-channel, which might or (more probably) not be available in Iron, depending on which version or Chromium it is based on. Some (all?) newer videos on Youtube use this format.

Other browsers with H.264 decoder are Safari, IE 9 and IE with Chrome Frame installed, and other browsers with WebM decoders are Firefox 4 and Opera from version 10.6. More on youtube html5 page.
