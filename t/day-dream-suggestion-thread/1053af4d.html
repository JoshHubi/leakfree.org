geekofalltrades November 24, 2018, 6:13pm #98
DansonDelta40:

Id also like to walk by default and run by using a secondary function, like the shift key. I dont want to speed through Black Mesa, I want to enjoy it! And it would suck to always have to hold down a key in order to do so. What about making the controls interchangeable, so you can choose whether you want to run or walk by default?

They are interchangeable, to some extent. You can use console cvars like cl_forwardspeed, cl_backspeed, cl_sidespeed, and cl_upspeed to change the default speed at which the player moves. If you want HL1 speed, you just up these values, and you sprint all the time! If you’re clever, you can even put them in a .cfg file, so you don’t have to do it manually every time you start the mod. And then source games already have a key for walking, don’t they? So you can use that if you want to slow down.
