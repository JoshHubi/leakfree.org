I don’t know how much help this is seeing how you’ve already done a format (meaning all your data is gone), but try going into the BIOS and resetting it to factory settings (usually F5 to load defaults and F10 to save and exit, but this might vary). Then boot from a Linux (Ubuntu) live CD to see if this works.

If the live CD works then you might be able to see what the problem is from within Linux, and maybe use MemTest (it’s on the disk) to check your memory.

Also, if you manage to get into Linux from the live CD, feel the fans on your PC to see if they’re running hotter than usual.
