JeffMOD August 11, 2020, 7:03am #11

If you want to be able to run HL from the editor, you’ll need to tell Hammer to run steam, and then set the parameters to run HL’s Steam ID with any additional game parameters you want (such as sv_cheats 1 and developer 1)

I also recommend The Compilator for final compiles (you can set a bunch of settings such as wadinclude, smooth and various other standard compile tool settings easily within the GUI) And SHLT for your compile tools (It’s an upgraded version of ZHLT)
