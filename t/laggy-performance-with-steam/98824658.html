Hi, I’m having this problem as well. I’ve been playing the game on full everything since it was launched and I’m up to the assassin part. I boot the game up this morning and, all of a sudden, it’s choppy as hell.

There’s a “known misbehavior” (i.e. a bug) that sometimes happen when you load up savegames for the first time after starting up BM - for some levels FPS drops down to the unplayable levels and stays there for a minute or two and then goes back to normal. It is yet to be investigated what does cause this slowdown and why does it affect only a number of players while most of us seem to be unaffected by it. Thus, try starting up the game, load up save and leave your PC alone for several minutes. Then try get back to playing and hopefully you would be free from this problem for the rest of your gaming session.

I’m having a very similar-sounding problem. I’ve been able to run HL2, HL2e1, HL2e2, TF2, Portal 1 & 2, and the HL2 Cinematic Mod series on my machine at max or near-max settings with no problems or laggy response.
I loaded the BM mod last night, and although it looks awesome, it runs like my graphics card is from the late 90s. Looks like I’m getting 5-15fps. I can’t aim, I run into doorjams, head hoppers kick my ass.

For me, it has run this way from the start, and does not improve after a few minutes (as has been suggested). It appears very similar to the kind of lag experienced when playing with inadequate graphics hardware, except that no matter how high or low I put the graphics settings, it doesn’t change.

I attempted both suggestions in this thread. Disabling the network connections had no effect. Running the game manually with the ‘-game “C:…\BMS”’ parameter switch resulted in a slight improvement but the game is still not playable.

I’m running Win7 Ultimate x64 with a GeForce 8600 GTS gpu on an nForce 680i LT SLI(C55P) chipset, and I’m using the latest graphics drivers for my hardware, according to nVidia; version 306.97.

Hoping this is resolved soon, I’ve been looking forward to this mod for a long time! ^o^

Update:
After trying the fixes listed in this thread, I also tried the fixes in this thread. Still no results. I have noticed that a couple of people have mentioned nVidia drivers might be the problem (if you’re an nVidia user anyway), and in the other thread someone recommended updating to the latest nVidia driver version that came out last month, as that solved all his lag/stuttering problems. Unfortunately, as I mentioned earlier, I was running the latest drivers. No fix for me there.
Later I read someone saying they rolled their nVidia driver back to a 29x.xx version, because the latest driver was giving them crappy gameplay. So I went to the nVidia website to download a legacy driver, and decided what the hell, I’ll try the latest Beta driver first. As of this writing, the latest beta is version 310.33. I downloaded it and did a clean install.
When I ran Black Mesa, it performed beautifully. No lag, no stuttering, I don’t run into door jams or get owned by head hoppers. So for any nVidia users out there, I recommend giving this fix a try.
