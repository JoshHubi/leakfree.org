I have the Release Preview, and I’ve been using Win8 since the Developer Preview.

The OS has a number of features that Win7 doesn’t have, which makes it nice (my favorite being the new Task Manager which actually comes in very useful).

But for the most part it is Windows 7 + Metro. It runs slightly smoother, and boots up faster than 7 (on my machine) but seems to have even more problems with backwards compatibility so far. I am hoping MS will fix that.

The Metro part is strange, and I’ve come to not use it at all. When my PC boots, I log in using my email address (it stores your settings and login information in ‘the cloud’), then the very first Metro Tile I click on is “Desktop”, and I do my best to avoid the start screen from there on out.

…Of course, you can’t completely avoid using the start menu always, and I’ve gotten pretty fast at searching for files and apps in the new menu.

The only Metro apps I was using regularly were [b]Reader /b, Pictures (.jpg, .png, etc) and Mail. I’ve replaced those with Adobe, Windows Live Photos, and Outlook.

If Windows Store begins to hurt Valve’s market in someway, I will be inclined to switch OS’s… perhaps Valve can start work on one…
