Crypt November 25, 2018, 4:02am #4

I’ll pitch in a little as well.

1a: You could also drop your video settings.

2b: Bhopping is still possible-ish, but it’s very difficult and the speed boost isn’t so great.

3b: It says Enable Iron Sights, but it’s only for the .357. The option exists solely because purists complained about that addition on that single weapon.

5: Because it runs on Source which is almost exclusively a Steam thing, and therefore requires Steam to run (like all Steam games). Run Steam in Offline mode.

7: This has no bearing on anything whatsoever, but if you really find it an issue, use the console command “sv_unlockedchapters 1”
