BaTz281 August 11, 2020, 5:17am #3

Russilker has the short answer for you.

A slightly longer answer - STOP USING THE COMPUTER! In the odd chances that any of his data is still intact, it will floating around the hard drive marked for deletion. This means that any time the system needs to write something to disk, it will choose Marked files before using free space on the drive.

If you, he, or someone you know is technical enough, pull the hard drive from the computer, hang it in another system and run something like Recuva, doing a deep scan for all file types. If you must leave the drive in the system, load Recuva onto that system and ONLY run that scan; do not load any other software. That’s one of your best chances for getting data back. If the factory restore zeroed the drive, well, he’s SOL and JWF.
