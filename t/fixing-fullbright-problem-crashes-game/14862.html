Flavrans November 25, 2018, 4:53am #3

If you are running the game on a laptop with a dedicated GPU and switchable graphics, make sure the game is running on the dedicated GPU (type [tt]mat_info[/tt] in the console to see which GPU the game is running on).

Otherwise, try disabling CSM (Cascade Shadow Map) by setting “Shadow Detail” to medium. Or type [tt]cl_csm_enable 0[/tt] and [tt]cl_depth_enabled 0[/tt] in the console.
