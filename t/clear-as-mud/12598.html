danielsangeo August 11, 2020, 10:37am #16

First, chill out.

The MD5 downloaded fine for me. As for the torrent, you need a client that will download from torrents.

The following are what I used in Windows with success: BitTorrent, uTorrent, and Vuze. Download one of these programs and then open the .torrent file in it. The program will then begin searching for “seeders” (those people that are connected to the various “swarms” (the hubs that your computer connects to as well)). Once it finds people seeding the file (allowing it to be uploaded from their computer and downloaded to yours), it will begin downloading (it could take some time for the torrent client to begin downloading especially if very few people are seeding at the time. However, once you connect to a seeder, you will start to see “pieces” downloading, small chunks of data at random. The more seeders, the more chunks you can get at any one time as a proportion of your internet speed). Be patient if it doesn’t start downloading right away; it will start downloading soon.

If a lot of seeders are available, it should go pretty quick, depending on your connection speeds. Once done, you should have three files in the torrent client’s download directory: blackmesa.7z, blackmesa.md5, and blackmesa-setup.exe

Make sure that all of the three files are completely downloaded, completely exit Steam, and then run blackmesa-setup.exe (which needs the MD5 and the 7z file to work–you can’t do anything with the MD5 file yourself; the setup program must use it) and go watch some cartoons or an episode of Breaking Bad or something because it’ll take a while to install. Then load up Steam and “Black Mesa” should be available in the list of games.

Let me know how it turns out.
