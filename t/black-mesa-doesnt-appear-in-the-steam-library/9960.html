Katana November 25, 2018, 2:19am #8

Steam looks for a gameinfo.txt file in Steam/steamapps/sourcemods/modnamehere/gameinfo.txt. That’s what it uses to add it to the steam list of games. Also for the record, the Black Mesa folder needs to be named ‘bms’.
If Steam isn’t detecting it at all, that may actually be an issue with Steam, rather than the game. Just having a properly-formatted gameinfo.txt in that folder should cause something to be listed.
