This is a result of a problem that (as far as I know) has been in the game since day 1. When you go back to 5e, many of the entities will now be missing because they don’t transition back to 5e when the player does.

Just load a save that was before bm_c2a5d, or start a new game on the map itself (bm_c2a5e)
