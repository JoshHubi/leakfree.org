xWhiteDarkness November 24, 2018, 6:37pm #6

You need to create 2 path_corners (or path_track if you want, I like the corner one more). IN THE CENTER of your train. Try to navigate with the 2D views to get the path_corner in the precise middle of your train. Now create another path_corner, at another point (and so on). Select the first path_corner and press ALT+ENTER. Name it Track_1 and next stop target Track_2. Go to the second path_corner and do the same thing but name it Track_2
Now go to the train and press ALT+ENTER, name it something (if you want a button to activate it) and choose First Stop Target to Track_1

Should work now. You can ask me any question about Hammer, quite good experience with it.
