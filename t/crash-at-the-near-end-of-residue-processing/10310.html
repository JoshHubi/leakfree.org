I had the same problem. If you don’t mind missing the very end of “Residue Processing” you can do what I did.

Go to ‘Settings’, ‘Keyboard’, ‘Advanced’ and activate ‘Enable developer console’.

Press the ` key whilst in game, type ‘sv_cheats 1’, to activate cheats. Then type ‘MAP c2a4h.bsp’ and it will load the start of “Questionable Ethics”.
