Mangled November 24, 2018, 7:07pm #8
garthbartin:

This seems fairly simple but something I don’t know how to do, so could somebody please help me here? I’d be happy with just knowing how to add curse to my game configurations.

Open Source SDK, select the version of Source you want to use in Engine Version (you will want to use Episode One for HL2DM)

Then go to Edit Game Configurations and then click Add, type in Curse for the name and set the directory to steamapps/sourcemods/curse.
