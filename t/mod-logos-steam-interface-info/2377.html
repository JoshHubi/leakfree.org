Before this got purged off the steam forums, I wanted to throw it someplace else. I know I wasn’t the only person that was bugged by this, but in short I figured out a way to rename and re-logo the mods that show up in your steam list.

Add a Steam interface logo to your Half Life mods
Create a logo.tga picture that is 16 x 16 and put it in the main directory for mod. Example: C:\Program Files\Steam\steamapps\acct\half-life\asheep
In the same directory, find and open liblist.gam file with notepad and add this line icon “logo”

Edit your Steam interface game names
Half Life Mods:
Go to the main directory for mod. Example: C:\Program Files\Steam\steamapps\acct\half-life\asheep
Open liblist.gam file with notepad and edit the line game “Azure Sheep” to reflect how you want the name to show in the steam interface.

Half Life 2 Mods:
Go to main directory for mod. Example: C:\Program Files\Steam\steamapps\SourceMods\CausalityEffect
Open gameinfo.txt file with notepad and edit the line game “Causality Effect” to reflect how you want the name to show in the steam interface.
