Level November 24, 2018, 5:47pm #2

Running programs off a shared HDD isn’t a great idea. It’s a much better option to just store files there.
If you have a wireless network, it’s most likely due to a the network connection cutting out. Most of the time this can go unnoticed, but if your computer is accessing files, then it cuts off half way through, that’ll cause issues.
If it’s a hard wired connection, I don’t know what it would be.

Changing the server machine to linux probably wouldn’t help if you’re running windows. It’s just cause more incompatibilities, and sharing files wouldn’t be as easy (depending on what you use).
