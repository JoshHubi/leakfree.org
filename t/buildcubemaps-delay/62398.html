Buildcubemaps Delay

General Development Source 1
Admiral_Sakai July 3, 2019, 4:33pm #1

When running the ‘buildcubemaps’ console command to, well, build cubemaps for the first time in an execution of the game, the game will hang for up to several minutes before starting the process. All subsequent attempts to build cubemaps start immediately. This was not occurring before the Xen Beta.
