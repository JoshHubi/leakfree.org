You can create the normal model within that poly budget, then you can add a turbosmooth modifier. You can turn it off at any time, so when you render you turn it on but when you work on your model you turn it off.

Keep in mind that using subdivision means you have to modify the model slightly by adding certain ring / loop edges otherwise you will lose your sharp corners. The end result is totally worth it. These extra edges can be removed easily by using the same ring / loop method method to select them and remove them.

Never use chamfers for this, they cannot be reversed.
