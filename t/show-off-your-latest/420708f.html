The texture name you apply to the model when you export for compiling must be the exact same as the texture to be applied in-game…the QC defines the path for materials (vmt) and the VMT defines where to look for the actual texture file (vtf).

Edit: MDL Texture Info tool will display the path and filename of the texture it is looking for so you can place/rename yours accordingly.
