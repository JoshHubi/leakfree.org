johnyo November 24, 2018, 10:10pm #8
  1. reboot PC
  2. enter the BIOS
  3. look around for a submenu containing “CPU clock”, “Vcore”, “Memory multiplier” and stuff like that.
  4. In there enable custom CPU clock and set it just above the default
  5. save and exit
  6. if PC boots fine, increase the clock some more. If it doesn’t, clear CMOS, repeat step 4 and up the Vcore voltage a bit. Search for maximum recommended OC voltage for yor CPU and keep it under that.
  7. Report progress.

[EDIT] just for the record, I don’t know jack shit about OCing
