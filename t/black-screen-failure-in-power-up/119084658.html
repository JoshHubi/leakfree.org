Forum search could be your friend. Back in the days the game had been released I had been investigating this bug and had come to a conclusion that with older GPUs like Radeon 9500 (i.e. pre-SM2.0 DX9 hardware) or with some crappy Intel “integrated GPUs” the game fails to render fullscreen noise shader properly resulting in totally black screen. Due to a bug in the game engine (in BMS game server dlls, not in Source engine itself) is it impossible to disable noise shader using console variable - it seems that it is simply ignored by the game. But it was possible to kind-a workaround the problem by messing up with a game shader dlls and definition files. I can’t recall exact details now but you could try reading this thread for a possible solution:
https://forums.blackmesasource.com/showthread.php?p=502093#post502093
https://forums.blackmesasource.com/showthread.php?p=502137#post502137
