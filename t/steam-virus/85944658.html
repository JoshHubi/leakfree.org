Jim_DiGriz November 25, 2018, 1:34am #23

To be absolutely sure (well, as absolutely as you can get without wiping your drive(s) and reinstalling), go to safe mode, and run Housecall from Trend Micro. I’ve found multiple viruses on computers with that, and they weren’t false positives. It’ll take a while, so start it before you go to bed, then check it in the morning. If it finds anything, clean them and run it again, just to make sure the other viruses/trojans/etc… werent masking others.

Jim_DiGriz November 25, 2018, 1:34am #24

Oh, forgot this: When you get Housecall up, go into the settings and make sure you run a complete scan. It defaults to a quick scan, which might not find deeply embedded stuff.
