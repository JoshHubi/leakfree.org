[WILDSPECULATION]

While I don’t know the current performance of the mod, I’m going to make some rough guesses. The FAQ states that if you can run Episode 2, you should be able to run BMS. I’m guessing that with the vast areas in BMS, combined with the amazing levels of detail and models, you should expect a drop of 5-10 FPS. While the team has likely done/is doing a lot of optimization, the large number of models and supposed size of the levels still creates a performance hit. If you need the extra FPS, there is, of course, always the option to switch to DirectX 8 and turn down some settings. Removing eye candy such as anti-aliasing and HDR, as well as water effects.

TL;DR: If you can run EP2, you should be able to run BMS with only a minor perfomance hit, but you should be able to regain your old speeds if you turn down settings.

[/WILDSPECULATION]
