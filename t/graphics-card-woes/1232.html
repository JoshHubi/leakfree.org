It has been a while since I’ve had any hands on with hardware that old, but that could very well be about what to expect. It may be more your CPU limiting you than anything (though running AGP these days is something I have to think back about so far as performance capabilities).
Single core 2.66 P4 is probably your biggest bottleneck at this point.

A 3.0GHz core2 dual core processor with a 3850 can run TF2 at about 35FPS when pumped up to 2560*1600. So the card has the umph to run the game, but your CPU doesn’t.
