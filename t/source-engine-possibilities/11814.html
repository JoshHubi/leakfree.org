Valve have a NDA on licensing their source engine. However, you can create anything you want using source and release it for free. If you buy the license I assume you get more access to the code but I do not know, either way you can code with the free version too.

They have a big list of source engine capabilities on their website which I think they recently revamped:
https://source.valvesoftware.com/index.php

So yes, standalone is possible with the license.
