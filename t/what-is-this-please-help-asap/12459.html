I’m the noob here, but this is definetly a bug. Probably something with the engine / water effect. Others can surely explain this in greater detail.

  1. Quit and reboot. Start game from a different map, seeing if this bug still happens.
  2. Check your graphics settings.
  3. Update your graphics card’s driver.
  4. If this problem emerged after you installed some kind of mod for Black Mesa, consider removing it. (Edit: backing up files before overwriting them is usually recommended even by the modders)
  5. If all else fails. Reinstall Black Mesa.
