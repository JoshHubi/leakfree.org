Fullmetal99012:

My Lenovo ThinkPad has an SWSetup folder, it has the preload software and drivers in it. usually driver upgrades extract there. Maybe HP has an updater service that runs in the background.

That would be my guess as well. As CPU said it’s an HP folder, it’s the default location for driver package extractions. If it’s happening automatically, it’s probably an automatic update service that’s doing it, or else maybe you tried to install some driver and it failed and is now trying to do it again. If you go into the SWsetup folder and look at which subfolders have what creation/modification date, you might be able to track down which package it’s trying to install (the subfolder name is the same name as a driver package from HP).
