Black Mesa Public Beta Patch - Jan 23rd

Black Mesa Archive Patches
Fr0z3n August 11, 2020, 11:29am #1

Hey guys, we’ve released a Beta Patch this afternoon for the Black Mesa Public Beta. Ontop of the previous fixes from the previous beta patches. The notes are as followed:

  • Fixed extremely long loading times on game launch. Now, it should launch as quick as, if not faster than, the main version of the game
  • Fixed game_round_win Entity to actually win
  • Fixed bouncing Bolts from the crossbow in MP
  • Fixed a bug in the balanced spawn system
  • Fixed Shotgun Collision model
  • Fixed various spelling errors and missing .wav files in dialouges.
  • Fixed the Crossbows lighting origin
  • RPG rocket won’t clip with the rocket when reloading.
  • Dr. Kleiners head and hands are now properly smooth (smoooooth as jazz)
  • Fixed blood not splattering on the MP5
  • Fixed the where placing 11 satchels crashed the MP server
  • Disable Muzzleflash option will not properly work
  • Various fixes/tweeks to Crossfire

Things we know about (this list is actually much longer, but I am listing the bugs that are the most known and/or we’re currently working on fixing).

  • Workshop Addon button in the main menu does not work
  • Grenade explosion time is 1second longer (We seriously don’t know whats happened).
  • Various things with the workshop still need work.
  • Various other things that I don’t have in front of me right now.
    Also: If you’re having issues with Missing textures or models and you have used mods (presently, or in the past) you will need to do the following steps to fix the missing asset problem:
  1. Unsubscribe from all Black Mesa mods
  2. Go to SteamApps\workshop\content\362890
  3. Delete all folders within this folder
  4. Restart Black Mesa, and you should be good to go.
    Once you do this, you should be able to re-subscribe to mods.

What was happening is that with the new system, there was some misdirection of filepaths based on the old system, that was leading to the missing asset problem. (It’s weird).

Anyways, thats it for now. This thread is for talking about these bugs, and these bugs only. If you have any additional bugs that you wish to report, please do so in the Crossfire Update - Known Bug List

If you don’t know how to use the Black Mesa Public Beta, read these steps: Right click on in your steam browser, click “Properties.” Go to the Beta Tab. Select “Public Beta.” You’re now in the Public Beta! If you no longer want to be part of the public beta branch, you can go back to the Beta’s tab under properties, and set your beta to “none - Opt out of all betas.” When you opt in, you’ll see a download happening, that’s perfectly normal.

Thanks everyone with your help so far. We hope to be moving all these out of the Public Beta branch, and into the main game in the next few days.
