Lamsey November 24, 2018, 5:31pm #28

Given that Macs these days are glorified PCs with standard x86 parts (the only thing about them that’s really different is using EFI rather than BIOS), there’s no reason why they can’t run games well when running Windows. Boot Camp doesn’t do anything magical, it just emulates a BIOS so that Windows can boot up on an EFI system.

They’re not really designed for it, though - Mac usually have weak graphics cards; you have to buy a Mac Pro before you get a decent GPU, and even then it’s woefully underpowered for a workstation-class system. Using MacBooks (Pro or otherwise) for gaming is also usually a bad plan, as the fans in these systems are prone to failure if the system’s stressed for long periods of time.
