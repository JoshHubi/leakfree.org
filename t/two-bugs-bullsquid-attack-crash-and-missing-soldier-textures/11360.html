Two bugs: Bullsquid attack crash and missing soldier textures

Black Mesa Archive In-game Issues (Mod)
Sinndogg November 25, 2018, 2:42am #1

I’ve done various searches for information on this first bug, but found nothing. As the title says, every time I have been attacked by a bullsquid (meaning, whenever its attack actually hits me), I experience an instant CTD (“hl2.exe has stopped working…”). I assume this may result from a problem with the visual effect that’s supposed to be superimposed over my vision whenever this happens? In any case, I’ve attached the .mdmp file that was produced when it happened last.

Also, the second issue is that some soldiers seem to be missing/have glitched textures. For example, some of the soldiers’ helmets are just plain white, with only the lighting/shaders working on them, while I’ve seen one or two with blue hands (which looks like a normal map transparency issue). I’ll admit, I haven’t looked this up, but my main issue was the bullsquid bug, anyway.

EDIT: I checked the integrity of the downloaded archive (all fine), and my DirectX is currently at 9.0 (according to the game, whereas the missing textures are apparently problems in versions before that), so I’m thinking that it’s not a problem with either of those.
