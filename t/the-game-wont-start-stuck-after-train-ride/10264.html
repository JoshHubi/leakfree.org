Good news, i suppose.

The problem is Windows 8 and the fact that Steam software needs to write in Program Files folder. I solved it by running Steam as administrator, but I suppose when you install Steam you could place it somewhere else than C:\Program Files folder.

I guess when someone is running Win7 on user/guest account, it could end up with same issue.
