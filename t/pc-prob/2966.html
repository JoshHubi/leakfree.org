MaxTheLimit November 24, 2018, 7:57pm #11

I suggest taking it down to the most basic set up.
Remove the mobo from the case, and just include cpu, ram, and display adapter (if it isn’t onboard of course )
Hook up the PSU to the mobo, short the power pins and see if it posts.
A couple possible trouble spots is certain mobos won’t post unless the CPU fan pins are shorted, so either make sure the cpu cooler fan is hooked up to the proper fan header on the mobo, or short the pins.
If it posts, start adding components until it fails.
If it doesn’t it might have been something you missed.
Just a couple ideas
