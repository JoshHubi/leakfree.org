Jeannotvb November 24, 2018, 10:24pm #4

You can’t just buy a memory stick and put it in a computer then expect it to work properly.

If the PC doesn’t boot then the memory is probably not supported by the motherboard, otherwise check the BIOS to see if the memory is recognized. If it is recognized by the BIOS, make sure to enter the proper voltages and memory timings as the computer will not do this automatically.

I suggest you contact the manufacterer to find out which memory the laptop supports, or look it up on the internet.

This. Laptops are especially finnicky about RAM. Depending on the model, some can’t even be expanded beyond stock - if it came with 1GB, it’s dying with 1GB. Others only allow you to add 512 (which is retarded, since you’ll lose dual channel if you run mis-matched RAM). Finally, you have to check the DIMM slot, voltage, timing and capacity as per the manufacturer, like Jeannotvb said.
