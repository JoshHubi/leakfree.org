The only problems I experienced was connecting to some servers (it said a server was full even if it wasn’t), and actually beginning a match when I was in one (it either didn’t start a countdown or went down to 1 and stuck there). I also noticed various bugs like the server ping not displaying correctly, and there being a long delay between sending an IM in the server chat box and seeing it come up.

In-game, it was shockingly like Modern Warfare multiplayer. The nanosuit mechanics definitely added some enjoyability, and the invisibility/armor mode are well-balanced, giving you a sense of power when activated without being too powerful. I enjoyed MW multiplayer (probably because aside from counter-strike and HL2 deathmatch its the only FPS multiplayer I’ve ever played), so this was still fun for me and I did pretty well, but I would’ve liked something new.

The graphics were definitely disappointing. I’m not really a graphics whore, but I expected a lot more from Crytek. Hopefully the single-player DX11 mode won’t disappoint.
