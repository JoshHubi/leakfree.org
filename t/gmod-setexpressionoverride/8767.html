New source is stupid about VCDs because Xbox is stupid about small text files. Any VCD files moved into a new game directory must then be compiled as part of its “scenes.image” file. To do this, open up Source SDK for the game in question, then go to File > Rebuild Scenes.image. You may be able to move this file between some Source games.
