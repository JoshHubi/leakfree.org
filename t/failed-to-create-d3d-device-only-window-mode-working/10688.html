lexa2 November 25, 2018, 2:27am #2

You’re not the first to report this problem. It seems to be Optimus-related drivers issue. No fix (except for playing in windowed mode) known so far.
You could “workaround” it by using borderless windowed mode. Add following to BM command line parameters:

-windowed -noborder -x 0 -y 0 -width <your screen resolution width> -height <your screen resolution height>
