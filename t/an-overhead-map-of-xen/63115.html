Hi all,

I’m here today to present to you some new overhead maps! This release features the entirety of Xen, Gonarch’s Lair, and Interloper. Here they are:

561×800 169 KB

858×800 247 KB

These have been a long time coming. I should note that Xen is separated from Gonarch’s Lair / Interloper as they are not physically connect; however, it is possible to calculate an exact displacement between the two areas (if one were to look at the skyboxes), which I have not done.

I’ve also created a zip file that contains higher-definition versions of these maps as well as overhead maps for each and every map of the game. It is here:
https://www.dropbox.com/s/gybj91i8y4lahln/Black%20Mesa%20Overhead.zip?dl=1

I will note upfront that the individual maps are oriented as they are in-game. For composite maps, Overhead C (which is the Xen area) has been rotated 90 degrees so that it faces its true orientation. Likewise, the tip of Nihilanth’s tower has been rotated with respect to how it is mapped in the game so that it too faces its actual direction.

Enjoy!
