jerry:

yeah, and it is possible, i just cba to do it, i rarely ever take screenshots (btw Namoroka is a codename for Firefox for Linux, so that was a webpage i had open)

You could just bind some kind of screengrabber (eg. ksnapshot, if you’ve got it) to keycode 107. In my fluxbox config I’ve got this simple script bound to 107:#!/bin/bash DATE=`date +%Y-%m-%d` TIME=`date +%H-%M-%S` if [ -f ~/snapshots/snapshot-$DATE.png ] then exec import -window root ~/snapshots/snapshot-$DATE-$TIME.png else exec import -window root ~/snapshots/snapshot-$DATE.png fi

Also, Namoroka was just the development codename for Fx 3.6.
