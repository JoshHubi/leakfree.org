Check your Windows Update history if you haven’t already - on one of my family’s computers the update had automatically started and failed several times without displaying any notification or error message.

Ended up having to do this:

varg:

Delete every files and folders in C:\Windows\SoftwareDistribution\Download
And then open up the command prompt under administrator; type wuauclt.exe /updatenow and see if it works for win 10.

It started a ~2.7GB download through Windows Update, which finally enabled installation of Windows 10 once it completed.
