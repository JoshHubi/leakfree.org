LeXX November 25, 2018, 2:18am #32

Hi 2 all lovers of HL!
I’ve got this problem too and nothing written here could help me!

But I [COLOR=‘Red’]SOLVED this:

  1. Firstly, create folder [u]episodic[/u] in yours source sdk base 2007 directory: \Steam\SteamApps%profile_name%\source sdk base 2007[b]episodic[/b]
  2. Then, create folder [u]bin[/u] in created folder: \Steam\SteamApps%profile_name%\source sdk base 2007\episodic[b]bin[/b]
  3. At last, copy server.dll and client .dll to the last created folder from \Steam\SteamApps%profile_name%\source sdk base 2007\sourcetest\bin, BUT NOT FROM BMS

Good luck & have fun!!! :freeman:
