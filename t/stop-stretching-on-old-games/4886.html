FireTime August 11, 2020, 4:46am #2

the wide screen hack you have works, It’s explorer that is messing up the colors. The problem is that when the computer switches to 256 color mode windows explorer still reserves a few colors, thus overwriting those allocated to starcraft.

Create a bat file in the same directory as starcraft and sue this code

taskkill /f /IM explorer.exe
Starcraft.exe
start explorer.exe

also if that hack isn’t working the scripts/hacks on widescreen gaming forum are usually good.

Grey_Acumen November 24, 2018, 10:00pm #8

I usually just right click on my desktop, then select NVIDIA control panel from the dropdown list. Once the NVIDIA control panel is opened, look for an option called “change flat panel scaling” and set that to “Use NVIDIA Scaling with Fixed-Aspect Ratio”

You’ll obviously need an NVIDIA graphics card, as well as the drivers to support it, but if you can get it working, then it’ll be a lot easier than all these other suggestions, and work for other programs that also have widescreen stretching issues. I know you mentioned wanting to use this method, so hopefully that’ll help guide you to finding that option.
