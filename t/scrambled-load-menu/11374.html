It appears that Black Mesa was unable to load the Resource (.res) file from one of the .gcf containers.

I tested this with a blank .res file in Source SDK 2007’s directory - without results; placing the empty file in \SteamApps\sourcemods\BMS\resource, did have an effect, however (similiar to yours).

A possible fix for this could be:

  1. Close Black Mesa.
  2. Download the attachment “LoadGameDialog.zip”, from this post (virustotal Scan Details below).
  3. Un-zip it to a location where the contents are easy to find.
  4. Copy “LoadGameDialog.res” (from “[b]…\LoadGameDialog[/b]”) and paste it into this directory: “\SteamApps\sourcemods\BMS\resource”.
  5. Set the file to “Read-Only” (Right-Click on the file -> Click Properties -> Check the Read-Only checkbox -> Click OK).
  6. Start the game and see if it had the desired effect(s)!

Note: Replace with Steam’s location on your system ([b]D:\Program Files\Steam[/b], for example).
