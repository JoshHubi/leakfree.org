That’s a limitation based on the old “predisaster” flag that set dialog contexts and prevented NPCs from following the player.
Gearbox just didn’t fix it for the intro because it was too low a priority for something that would require coding an additional flag into the AI.

Since Black Mesa doesn’t use quite the same code architecture, and in Source it’s easier to set use lines on a case-by-case basis (think the scientist in the AM locker room), we’re not bound by that.
