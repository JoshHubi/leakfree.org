Crypt August 11, 2020, 10:47am #2260

Don’t worry about interrupting things. We’re organized now and do our heavy dev-discussion privately. Go wild with the thread, that’s why it’s here!

Anyway, we’re just trying to add some bits for it to seem a little bit like a prologue to the main game. If you notice, the original HL’s HC was a simple training level. Just some pretty rooms with instructions. The PS2 version added an introduction to make it seem more grounded in universe (as opposed to solely existing for training purposes). And we wanna take that a little bit further, with an outro and expanded intro, and even an attempt to make the Hazard Course’s origins more clear.

We’re all avid HL fans, so we don’t want to meddle with canon too much, but obviously we can’t be completely grounded what’s known if we want to do anything more than the original PS2 intro. We clearly don’t want to mess with anything already regarded as fact/established canon, so don’t worry about that. We even went so far as to ask Half-Life’s writer when the Hazard Course actually happened.
