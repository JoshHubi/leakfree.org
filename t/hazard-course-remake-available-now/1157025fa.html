That would be both an animation thing and a choreography thing.

Choreographers can’t create animations. They have to work with animations that are pre-set by animators, which means the guard model would have to be recompiled with new animations if you want him to hand you a weapon, for example. Unless there already is one that the BM devs created and used, but I don’t recall that sort of animation existing.

Plus the Glock system is a nice automated way of doing something that the Hazard Course team likely doesn’t want to keep doing themselves. Up to debate. There is a lot about the idea that could use tweaking.
