This is a solution from an other topic with the same problem

DerKrauthammer:

I fixed BoosterGold’s issue, it should theoretically work for you as well.

If you are running Windows 7 and you have Steam installed to “Program Files”, you’re going to have to uninstall it and reinstall it to somewhere else, preferably C:\Games

Windows 7 locks down any applications installed to Program Files and it’s stopping Black Mesa from accessing the static player information needed to carry weapons over. You should be good to go after that.

For me this solution worked.
