lexa2 August 11, 2020, 9:26am #31

Steps to troubleshoot:
0) Remove any existing 7zip installation you have.

  1. Install latest 7zip from https://www.7-zip.org/ matching your windows version.
  2. Run 7z File Manager, navigate to where you had downloaded the game and run integrity check on the archive. It should pass without errors. If not - try running it once again.
    3a) If both first and the second attemps to test BlackMesa.7z integrity failed with an error - most probably you have a corrupted download. Easiest way to fix - use torrent to redownload and tell it to download to the same folder you have your corrupted files stored in. In such case it would only redownload the corrupted parts saving you from having to redownload entire 3GB archive. After torrent would finish downloading - repeat step (2) - i.e. test the archive once again with 7zfm.
    3b) If you have second testing attempt went ok while first had given you an error: check your PC. It is either viruses/trojans, faulty memory or overclocking-related problems.
  3. As soon as 7zfm would state that the archive is not corrupt - proceed with executing BlackMesa-Setup.exe. It should work fine now. If not - report back including all the exact steps you took before getting the error. Also don’t forget to provide the complete log of the installer output.
