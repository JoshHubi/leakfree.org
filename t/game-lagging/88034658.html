  1. Pull out your motherboard manual and read precisely what order the memory needs to be installed and verify that you are doing it correctly.
  2. Check your virtual memory settings (page file) set it correctly then reboot for it to take effect.
  3. Run memtest to verify memory integrity.
  4. Check out Microsoft’s performance options. https://support.microsoft.com/search/default.aspx?query=How+to+set+performance+options&catalog=LCID%3D1033&mode=r
