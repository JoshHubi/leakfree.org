higgsboson November 25, 2018, 12:41am #3

If the file you’re trying to save is too big, try to save it as a PSB, instead of a PSD. If that doesn’t work, the file is just simply too big and there’s nothing you can do except reduce the image size (delete some layers, don’t use Smart Objects, make the Image Size smaller, etc…).

Or it could be some system bug, if your file isn’t particularly big.
