I’m guessing you’ve already done this but have you set them to run as administrator and set the compatibility mode to either Vista and XP?

EDIT: I’ve done some reading and it turns out that the “MUA protection system” AKA DRM system that Actionvison employs on those games are not compitable on windows 7 and there is no work around that I can find.

Your two option that I see is to either get the virtual XP that comes with windows 7 ultimate installed or go download an unauthorized copy that has its copy protection removed. It be important that it be removed and not just patched because you will just run into the same problem.
