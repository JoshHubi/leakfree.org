I don’t know if it has been done yet here, but as I recently created my personnal configuration for Half Life which gives me satisfaction, I thought it would be a good idea to open a thread and share it, and share yours if you have one.

So the idea is to change, by the console commands, the damages caused by some weapons and the health of some ennemies. Of course, doing that manually each time you launch the game would be very annoying, so the commands are packed in a cfg file.
Just put the file in C:\Program Files\Steam\steamapps*user*\half-life\valve and, each time you launch the game, you only have to load a map,
type “exec name of the file.cfg” in the console, reload a map and that’s it.

This way you can drastically change the gameplay. Personnally I missed a “hardcore” mode for HL: in the hard mode the ennemies could just withstand a large amount of damage, which I find unrealistic and not very fun. So I made my own “harcore” mode, in which I increase a lot the damages made by conventional weapons and alien attacks, as well as the aliens health. I also drastically reduced the health given by medipacks and health chargers.
As a result, I find the gunfights more intense and more realistic (especially against the human grunts).
That, plus forcing myself to quicksave only one or two times per level (depending of the length of the level) greatly improves the game experience.

I give you my cfg file if you want to test it:
https://www.mediafire.com/?wqx4et84w0t6m10
