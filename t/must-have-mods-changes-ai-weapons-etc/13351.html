I disagree with the houndeye bit, because they were fodder in the original but are an actual credible threat here.

As far as the HECU goes, there is a setting where you can change that for now.

ai_reaction_delay_alert “0.3”
ai_reaction_delay_idle “0.75”

I believe you can type this into the console and it will save it. That, or it’s either skill.cfg or config.cfg that you can paste this into.
