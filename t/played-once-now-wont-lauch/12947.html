FERGUS November 25, 2018, 3:54am #2

It seems as some people have recently been experiencing problems while trying to play third-party mods on steam, Valve probably broke something again.

To rule out this possibility, it would be great if you could try to start any other third-party mod if you have any.
You should also try to verify integrity of the file cache for the Source SDK base packages. To do this, go to the tools section in your steam library tab, right click on the corresponding Source SDK Base(I’m not sure which one Black Mesa uses, it’s either 2006 or 2007), and go to properties. There you choose the local file tab and click on "Verify integrity of tool cache. When the check have been completed, try to launch Black Mesa.
