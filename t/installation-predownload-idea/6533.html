I know this if for ingame suggestions but I figure its the best place for this.

Just thought I’d suggest this idea that kinda came about in the “Got a Question?” thread (https://forums.blackmesasource.com/showthread.php?p=343434#post343434).

An encrypted pre-download pack with a unique algorithm would help reduce the end download size, create easy distribution, & eliminate leaks. In the download pack files could have invalid file extensions and/or directories removed to ensure that if the encryption key was ever discovered the contents are useless because of no file extensions or directories. Having a custom installer would be able to have the option to select the pre-download file and decrypt it (and/or correct file names/directories) to aid in a quicker installation.

My ISP where I live messes with torrents and slows large file downloads combat piracy. So for me having to download 3 gigs of compressed data could take well over an hour.

The prepack could be released when the release date is released(way to much use of the word release!). If others are in the same situation as I mentioned above this could make BM easier to access on release day & lessen strain on servers.

Just a thought I wanted to share. Thanks for reading.
