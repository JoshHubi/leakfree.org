You could check if an older save is still present in Steam Cloud, where you can download it:

https://store.steampowered.com/account/remotestorageapp/?appid=362890

If Black Mesa is still running, don’t quit it yet, or the older saves will likely be deleted/overwritten in Steam Cloud.

This only applies if Steam Cloud synchronization is enabled for Black Mesa (which is it usually is unless you disabled it.)
