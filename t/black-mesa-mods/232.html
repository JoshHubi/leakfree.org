Actually, I mentioned in the former thread that the scientists could utilize various experimental tech (that happened to have hazardous effects) that would cause a lockdown in a certain room (presumably with the marines caught in there already) and vape whatever is in there (much like the room in the Lambda Complex with the headcrab alleys, you could steal away to the operation terminal and annihilate them with electricity).

I figure the terminals could be operated much in the way they were dealt with in Star Trek Elite Force 2 (mini puzzle game) or probably more familiar to everyone Bioshock 2.

Scientists would have different access levels for different terminals/retinal scanners. No weapons (except for perhaps a pistol), but they would have the ability to activate turrets or other security systems.

Security Guards I figure would only be able to carry the pistol or shotgun, supplementary abilities I’m not so sure about yet.

Was thinking about a loose class system for the HECU as well.

Standard Grunt: Shotty, SMG some frags (can run faster than the rest).

Computer Specialist: Pistol and Shotty, also has the ability to hack terminals & retinal scanners. This would provide some fun moments where you have a scientist on one side of a retscan door and a CompSpec on the other trying to race each other to solve the puzzle.

Still not so sure about this one, but I figure there should be above ground areas to traverse, so I think a Sniper should be included.

Not so sure about a demolitions expert.
