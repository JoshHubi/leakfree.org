Try moving the folder ‘SAVE’, located in ‘\SteamApps\sourcemods\BMS’, to a remote location (such as your Desktop, another folder, an external HDD, ecetera), then starting the game.

If that does not work, create a new ‘SAVE’ folder in said directory and make sure its attributes are not set to Read-Only - after you moved the previous folder.
