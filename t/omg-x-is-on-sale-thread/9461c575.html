geekofalltrades November 25, 2018, 3:36am #115

First of all, each of you should grab FOSE/NVSE - they’re the script extenders. A lot of other mods depend on them, so they’re good to have. The first essential mods that spring to mind for me are the Sprint mods. There’s no sprint in vanilla FO3/FONV. These mods add a sprint function that consumes Action Points.

You’ll also want to Unofficial Patches. Beth RPG’s are notoriously buggy, and these community patches fix a lot of issues. Mission Mojave is the NV unofficial patch. The FO3 patch is just called Unofficial Fallout 3 Patch.

Another essential is a UI mod. I recommend MTUI. It has versions for both FO3 and NV.

I also really like A World of Pain. AWOP is for New Vegas, but there’s also a Fallout 3 version called AWOP4FO3. These are dungeon packs - they add a ton (in NV, I think it’s over a hundred) of new dungeons that are really challenging and really well-made. If you want mostly a vanilla experience to begin with, then save these for later, and they’ll be a great reason to replay.

Fellout: There are versions of this for both NV and FO3. It’s a really badass weather and lighting mod. It makes nights really stupidly dark, though; there are third-party fixes for that (see Fellout Nights Restored).

If you want snappy texture packs, a guy named NMC has done huge texture packs for both FO3 and NV. Search for NMC’s Texture Pack.

Enhanced Night Sky has versions for both FO3 and NV. It’s a night sky texture replacer that makes nights really jaw-dropping.

New Vegas has weapon modding, while FO3 doesn’t. For New Vegas, you should get the Weapon Mod Menu mod. It makes modding weapons much better and more complete. If you want mods in FO3, try Weapon Mod Kits. It allows you to add silencers, extended clips, and auto-fire mechanisms in various combinations to some weapons. It’s much more limited than NV’s modding system, but still pretty cool.

I recommend nexusmods.com as a great place to get all of the above mods and also explore other mods.
