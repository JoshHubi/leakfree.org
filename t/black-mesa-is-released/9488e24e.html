saivert November 25, 2018, 3:32am #699

It is not the complete engine that you get as a Source licensee. It it still basically Source Mod SDK but they now host it on Github. Don’t get your hopes up. You still need the Source SDK Base stuff installed from Steam to run games (this includes the binaries like hl2.exe and a host of DLLs containing engine code that you don’t get to tinker with). The Mod SDK lets you build your own server.dll and client.dll. That’s it!
