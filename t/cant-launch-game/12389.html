FRATS November 25, 2018, 3:36am #14

Hey guys i found a solution to the error you guys are having, i had the same error and i got it working.

Frist completely exit Steam

Go to your steam folder in Program Files (x86) for windows 7 (eg. C:/Program Files (x86)/Steam)

then go to SteamApps

then go to your account folder. (in my case its frats1)

Delete the folder Source SDK Base 2007

then go back and go to the common folder

Copy and paste the Source SDK base 2007 folder into your account folder (again in my case frats1)

Relaunch Steam again and try launching the game again.

For me this fixed it and im able to play again. Hope this works for your guys
