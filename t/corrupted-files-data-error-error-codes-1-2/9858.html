lexa2 November 25, 2018, 2:31am #11
spacehouse:

How does one make bittorrent redownload only corrupted parts of a 7z archive? i wasn’t aware bitT had that capability.

It is a native capability of bittorrent protocol cause it splits all the files you download into small parts and checks integrity of each part when you download it. You should place your BlackMesa.7z into a folder named BlackMesa and then instruct your torrent client to download BlackMesa.torrent to this folder. It would notice that you already have a file named BlackMesa.7z there and would initiate a so-called parts hash check procedure. Then it would proceed with downloading only the parts that don’t pass hash checksum check. Google on uTorrent user guide and usage patterns if you want more details, it is really a bit offtopic for BM bug reporting forum.
