TextFAMGUY1 November 25, 2018, 4:40am #2

You can change your FoV with the console command fov_desired. We made it so it can be turned up to 120.

The damage variables are now kept in the weapon file .dmxs, which are in scripts/gameplay/weapons, which you have to extract from the game’s VPKs. This is a more efficient way of storing the information, and prevents tampering for MP. But you can edit them freely as they are just text files.
