RabbiBob August 11, 2020, 9:22am #11

If you’ve installed to the default path and you’re not seeing it in your Library:

Make sure the filter at the top of the list is set to ALL GAMES.

HEADSMACK :fffuuu:

Otherwise, the following works to launch the game (alter for steam.exe install path).

“C:\Program Files\Steam\Steam.exe” -applaunch 218 -game “…\SourceMods\BMS”

or see this post: https://forums.blackmesasource.com/showpost.php?p=492357&postcount=9
