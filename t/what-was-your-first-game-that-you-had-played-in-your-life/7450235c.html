Also, to get it to work on W7:

"I found the solution to runing Fury3 in Win7.

I have only tried this with the Fury3 CD from Microsoft, I am not sure it will work from installed files on your old PC.

So. first the install, well to be more accurate the copy.

Copy the entire Fury3 CD to a folder under Win7, I called it Fury3 (no imagination)

Find Fury3.exe and through propertied set it to Window95 mode

Fire it up, when it complains about wanting something, get it from one of the Fury install folders and put it in the same folder as Fury3. I tried this a few times, from scratch to get down the instructions and it seemed to vary, one time it just worked, two others I had to copy stuff.

Make sure it is in Win95 mode or the joystick will not work."
