Thanks! Do you use some sort of mod manager or do you keep track manually?

Depends. As Pyro said, there are some that are only found on Nexus, but I’d say a good 95% of the mods I run come from the Workshop. Either manager is easy to use, though Steam is infinitely more so (you just subscribe to the mod and it’ll download and unpack next time you run Skyrim).

However, if you really want to unlock the true potential of mods, get the SKSE. It’s necessary for some mods which run tightly with the game’s engine (SkyUI is one such mod). It also adds a bunch more freedom in the ways of editing the aspects and options of different mods while in-game.
