The 3D selection bug is brought on by ATi not supporting that particular version of OpenGL properly. The drivers should work for Windows 7, but don’t hold me to it.
Also, for tuts on mapping, and some modeling and coding, check out TWHL. There are a few coders who might be able to give you tips, plus an archive of the old VERC tutorials, which have weapon coding. You’ll need to download the HL SDK and know a bit about C++ first, though.
