Neoplayer2 November 24, 2018, 9:38pm #12

In a more serious light, the GIMP is seriously the best image manipulation software I’ve used. I like it more than my copy of Photoshop Elements.

Use Script-Fu to make your own plug-ins, or get plug-ins and new filters and new brushes off the internet. It’s open-source, and under the GNU license, so it already has my thumbs-up. But the user interface is rather dodgy - you will probably want to wait until the GIMP 2.8 comes out, which will have huge UI improvements.
And you’ll want to get GTK 2.0 and some skins to pimp yo’ GIMP.

But really, it’s a powerful software that I use for image retouching, drawing, design…all of the reasons you’d want Photoshop.
