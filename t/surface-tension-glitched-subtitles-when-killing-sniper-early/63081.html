Surface Tension: glitched subtitles when killing sniper early

Black Mesa Archive In-game Issues
robot55 March 28, 2020, 4:14pm #1

In Surface Tension, before the “explosive building”, there is an area with snipers outside. The first sniper is hinted at by the dying security guard with a scientist running to his aid. The sniper is hiding in a ventilation shaft.

If you do nothing, the sniper will shoot the scientist. However, if you kill the sniper very quickly (e.g. with the RPG), the scientist will run off screaming. The German language subtitles are glitched in this case: not only the actual subtitles are displayed, but also some kind of debug message. Also, there is a typo in the subtitles (“Hife” instead of “Hilfe”). Please see attached screenshot.
