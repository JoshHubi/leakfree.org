a new linux update

Void-995;1729827777336153197:

Another revision has been pushed into public beta:

  • Removed any leftovers of static props lighting from shaders when it’s not used (possibly could resolve weirdly colored meshes on some configurations)
  • Tweaked shadow bias value for new dynamic lights to avoid shadow/surface Z-fighting (addresses tram lights issue)

As always, don’t forget to double check your game cache. Maybe even fully delete game (with any files left being deleted manually afterwards) and re-install it.

Be sure to find and delete file named glshaders.cfg inside of either /bms or /hl2. Sometimes this file stucks, deleting it will cause clearing shader cache, possibly resolving any issues left about memory consumption.
