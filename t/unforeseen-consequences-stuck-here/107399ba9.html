danielsangeo August 11, 2020, 11:07am #46

The attachment is not showing. I think it’s a faulty upload. If you’re referring to this elevator:
Image 1 | Image 2

… and that elevator is closed, you probably have found a bug because the doors are supposed to be open. Your options, if this elevator isn’t open to you, is either start a new game at the UC chapter or “noclip” into the elevator to continue.

To noclip (if you don’t know how that works), type sv_cheats 1 into the console, then type noclip. You will be able to pass through the doors. Type noclip again to turn it off and you should be able to press the button to move the elevator to the next chapter. Oh, and type sv_cheats 0 to turn off cheats.
