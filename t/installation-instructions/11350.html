Actually, this is one of the easiest mods to install, especially since it comes with an installer. I’m assuming you already have BMS downloaded. The download should have 3 files; blackmesa.7z, blackmesa-setup.exe and blackmesa.md5. After you have installed Source SDK 2007 (in the tools section of the library, just double click and agree to the terms), run the BMS installer. It will automatically verify the download and extract the files to your steamapps/sourcemods folder. Once it is finished, restart steam and BMS should be in your library!

Install Black Mesa:

  1. Be sure to install Steam (https://store.steampowered.com/Steam)

  2. Be sure to install Source SDK 2007. From inside the Steam interface click “Library”, find a dropdown menu that is default labeled as “All Games” and in that dropdown, choose “Tools”. Scroll towards the bottom of
    the list to find and install the “Source SDK 2007”

  3. Close down and exit Steam

  4. Go to https://www.gamefront.com/black-mesa-is-live-download-it-here/ scroll down to see three separate download links. Download all three. The complete game installation package consists of the three files:
    a) blackmesa.7z
    b) BlackMesa.md5
    c) blackmesa-setup.exe

  5. Install the game by clicking on blackmesa-setup.exe and be sure it is installed to this directory:
    \Steam\steamapps\sourcemods\BMS

  6. Launch Steam and find the game “Black Mesa” listed in your games library
