FAQ Thread:

MIRRORS: Mirrors, meaning in game reflective surfaces in which the player can see them self, and nothing to do with download locations, will categorically not feature in black mesa. This is because the current version of the source engine does not support them, and the coding necessary to make them workable is beyond the scope of this mod.
