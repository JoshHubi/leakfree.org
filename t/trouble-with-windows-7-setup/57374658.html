That’s why it won’t work… You have to copy the disk image entirely. What makes the CD bootable isn’t a file, so if you copy files with a drag-and-drop (or equivalent), it won’t work.
If you download the disc from a torrent or something, it should be a .iso, or some other disk image format. If you have an original DVD to copy from, you have to make a disc copy, and not copy the files themselves.

With PowerISO I think you can tell it to make the DVD bootable as well… If you don’t have any disk image source with you anymore.
You can probably Burn a disk image with PowerISO as well… never used that program much though.
