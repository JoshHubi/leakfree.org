Johnwalter August 11, 2020, 11:28am #2

It might be your save game.It happened to another user as well.
Every object (barrel, coffee mug, crate, etc.) that you can move is spawned when a map is loaded.
So naturally some objects can spawn incorrectly, causing a crash.

Simply try either:

  1. loading a save game before the last one.If failure again, load one before that one.
  2. If you have no more saves to load, start the chapter again.
  3. You can try loading the map in the console command, but that is for last resort.

To use the 3rd option:

  1. Open the Option’s menu, go to Controls, then click Advanced.Then check the Console box.
  2. Open the console command menu by clicking the cedilla key (its above the Tab and below the Esc key)
  3. Then write down for example “load bm_c1a0a” to load a map. Thread for all the maps in BM.
