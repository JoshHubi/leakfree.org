Joratze:

Windows 10 Pro, a clean, new install. The only programs currently running are Steam and the OS himself^^

  • Your AMD Phenom II X4 960T is overclocked and could be throttling, have you tried running it at spec speeds.
  • Whens the last time you blew out the dust bunnies.
  • Ram speeds can come into play, did you have to modify (lower) ram frequency for a stable overclock. CAS latency on 1333 DDR3 is sometimes not the fastest.
  • Are you running the latest drivers, and bios for the mb and gpu.
  • Windows 10 is still in it’s infancy, more like a beta OS imho. If you got time see how it runs under 7 or 8.1 which has mature drivers.
    For comparison I am running an AMD A10-5800k stock, GTX-660 stock with overclocked 16gb DDR3 1866 cas 8-8-8 on Windows 7
    It’s not an awesome experience with everything maxed so I decreased some settings. @1920x1080

Good luck on squeezing some more performance out of your setup.
