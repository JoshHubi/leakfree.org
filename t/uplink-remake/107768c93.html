I strongly concur and agree with the concerns expressed by Ulrik.

I just released a mod here on the BMS forums https://forums.blackmesasource.com/showthread.php?t=15636.
Although it does address errors within BMS, it’s most beneficial to those affected with hearing impairment issues.

My mod specifically updates the English language closed caption subtitles by correcting grammatical errors, sentence structure, and adding missing written closed caption dialogue content, so that what is audibly spoken by NPC’s is accurately reflected in the closed caption subtitles that are displayed on-screen. This is accomplished by correcting the “closecaption_english.txt” file and then reflecting those corrections by updating the “closecaption_english.dat” file. Both of these files are overwritten upon installing BMS Uplink.

Although you’ve updated these two files to reflect changes necessary for BMS Uplink, by overwriting my mod’s corrected files, you’re re-incorporating all the errors that exist in the original versions of these two BMS files.

Is is possible for BMS Uplink to be released as an independent and standalone mod, maybe in a folder named BMS Uplink, to keep it completely separate from BMS in order to avoid files being overwritten and potential issues associated with future BMS and/or mod updates? It would make things much simpler and eliminate a lot of headaches.
