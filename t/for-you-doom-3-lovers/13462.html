For You Doom 3 Lovers II

A group of individuals over at IO Doom 3 https://www.iodoom3.org/ have been tweaking the hell (get it) out of the Doom 3 source code that id Software released on November 22, 2011, in order to squeeze and wring out as much performance as possible out the id Tech 4 engine.

Their FAQ states “This project aims to build upon that source code release by cleaning up, fixing bugs, and adding features. Our permanent goal is to create the open-source Doom 3 distribution upon which people base their games and projects. We also seek to have the perfect version of the game engine for playing Doom 3, it’s expansion pack Resurrection of Evil, and all popular mods.”

Although they state that they have yet to release their mod, I came across a supposed release of IO Doom 3 on the Moddb website https://www.moddb.com/games/doom-iii/downloads/iodoom3

If you’re interested, try it out and see what kind of improvements/mileage you get. Remember to backup your original file(s)!
