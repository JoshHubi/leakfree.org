ketchup1001 November 25, 2018, 2:17am #16

I believe that game (Black Mesa Source) will only show up in your Steam games list if you install it to:
C:\Program Files\Steam\steamapps\sourcemods\BMS (which is the default directory the installer suggests)
I had mine installed to a different folder and couldn’t start BMS until I copied it over to the above directory.
You will also have to restart Steam after you install/move BMS. Make sure you have Source SDK 2007 installed before you restart Steam.

Hope this helps, cheers!
