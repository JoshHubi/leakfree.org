You have to get it to go online which should be automatic, although it doesn’t seem to go online for me very often.

Under “File”, hit “Offline Mode”.

Sometimes I need to leave it running for a while, sometimes overnight. Then go to “Available Games” and Mercs should be on the list. It will update to the latest version.

The alternative I mentioned is MXC (I think) which is posted in the MechWarrior 4 : Mercenaries forum. As I said, I haven’t tried it out myself yet.

You can torrent the MTX file for Mercs, copy it to the correct folder then use MTX to “Install it” if you have trouble getting it to go online (although i’ve never had to do it this way either).
