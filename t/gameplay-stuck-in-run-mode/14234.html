Fr0z3n November 25, 2018, 4:35am #2

Go to Options > Black Mesa and uncheck “Always Run”

The option wasn’t working before, but the recent patch fixed it and in the process of doing so made it on-by-default for all players. Whoops. I guess Gordon didn’t want to be late to work again.
