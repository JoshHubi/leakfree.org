Microsoft is officially unveiling the name for the next release of Windows today: Windows 10. While the software maker had referred to Windows 9 as codename Windows Threshold internally, today’s official naming puts any rumors of Windows, Windows TH, Windows X, and Windows One to bed. It’s simply Windows 10, marking a jump from the mixed reception of Windows 8.
